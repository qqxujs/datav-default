场景编辑器内的工具栏提供了编辑器切换的功能。通过该功能，您可以切换进入场景编辑器或蓝图编辑器界面。

## 使用方法

1. 在场景编辑器页面内 ，将鼠标移动到场景编辑器页面左上角的![蓝图编辑器图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0516197461/p415929.jpg)图标上方。
2. 单击即可切换进入到蓝图编辑器页面内，可进行蓝图编辑器页面内的各种后续操作。![切换到蓝图编辑器内](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3343458461/p415932.png)
3. 再次单击蓝图编辑器页面左上角的![场景编辑器页面入口](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0516197461/p415933.jpg)图标，页面自动切换至被选中的场景编辑器页面内，可继续场景编辑器页面内的其他操作。
