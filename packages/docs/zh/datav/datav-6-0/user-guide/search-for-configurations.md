  使用搜索功能快速定位修改资产配置-DataV数据可视化-阿里云

通过搜索资产配置功能，您可以快速定位到资产的对应配置项，在搜索配置页面直接修改配置样式，也可以复用同类型的配置样式。

## 操作步骤

1.  登录[DataV控制台](https://datav.aliyun.com/)。
2.  在我的可视化页面，单击您创建的可视化应用项目上方的编辑按钮。
3.  在画布编辑器页面，单击图层栏或画布中的某一个资产。
    
    **注意** 在搜索资产配置之前，请确保您的可视化应用项目中已经添加了相应的资产，否则需要首先添加资产，详情请参见[添加资产](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/add-a-widget-1/#task-2375434 "通过添加资产功能，您可以在DataV可视化应用项目中添加可视化资产，配置资产所需要的应用样式。本文以添加一个组件类型的资产为例，介绍在画布编辑器中添加资产的方法。")。
    
4.  单击编辑器右侧配置面板右上角的搜索配置。
    
    ![搜索配置入口](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4515178061/p53759.png)
    
5.  在搜索配置页面下方的输入框内，输入配置项的名称。
    
    输入完成后，按Enter键或单击右侧的![搜索](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8754713061/p174929.png)，系统会自动搜索符合条件的配置项内容（支持模糊匹配），并显示在当前搜索页面内。![搜索配置界面](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4515178061/p53760.png)
    
6.  在搜索配置显示配置项的列表中，您还可以修改和复制配置样式、定位跳转到配置项。
    
    -   单击选中某个配置项修改配置样式。![在搜索配置中修改配置内容](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4515178061/p53773.gif)
        
        修改完成后的配置会直接应用到可视化应用中对应的资产上。
        
    -   右键单击某个配置，选择复制样式，可复制资产的配置样式；右键单击一个或按住shift键多次单击选中几个同类型的配置项，选择粘贴样式，可将复制的样式应用到配置项中。![复制黏贴弹窗](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4515178061/p53764.png)
    -   单击某个配置项的目录栏或者右侧的![定位图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4515178061/p203190.jpg)，可跳转定位到该配置项所在配置面板中的具体位置。![切换定位到配置栏](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5515178061/p53771.gif)
        
        切换成功后，配置项的具体位置会以蓝色标记显示。![切换成功蓝色显示](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5515178061/p203192.png)
