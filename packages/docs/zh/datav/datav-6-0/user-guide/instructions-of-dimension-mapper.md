# 尺寸映射器使用说明 %{#289070}% 

尺寸映射器用于调整图元尺寸大小，通过使用范围滑块决定图元尺寸范围，本文以配置**基础平面地图 3.0**组件中**矢量散点层**子组件的散点半径为例，为您介绍如何自定义数据值和尺寸的映射关系。

## **配置说明**

配置地图子组件内散点半径，您可以通过两种方法调整散点的半径。

* 方法一：单击关闭**散点半径**配置项左侧的![关闭颜色映射](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4553257261/p260339.jpg)图标，在鼠标悬浮态提示**开启视觉映射**时，无需数据驱动，单纯的自定义样式配置，通过滑动滑块或者自定义输入数字，即可调整散点图形的尺寸大小。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1379881861/p633966.png)
* 方法二：单击打开**散点半径**配置项左侧的![打开状态](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4553257261/p260340.jpg)图标，在鼠标悬浮态提示**关闭视觉映射**时，使用尺寸映射器，由散点的数据驱动映射，可对散点尺寸使用范围滑动条进行配置。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1379881861/p633970.png)

## **功能说明**

|    **功能**    |    **说明**    |
|    ----    |    ----    |
|    面板折叠配置尺寸范围映射    |    在尺寸映射器折叠状态时，拖动左侧滑块调整最小值对应的图形尺寸；拖动右侧滑块调整最大值对应的图形尺寸。**说明** 系统会根据最大值和最小值的参数，自动计算出中间数值和图形尺寸后进行绘制。    |
|    面板展开配置尺寸映射细节    |    在尺寸映射器展开状态时，系统自动读取数据，进行图形尺寸和数据的映射匹配，您可进行如下映射细节的配置。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1379881861/p633997.png)**映射类型**：尺寸的映射类型，可选**名词分类**、**数值连续**和**数值分区**。详情请参见[映射类型](#2d144ba0deuk4)。**映射范围**：您可以拖动映射范围左右两个滑块或在下方自动读取的最大最小值对应数据输入框内，自定义输入最小值和最大值对应的图形显示像素的数值，控制图形映射显示的尺寸范围。**数量**：您可以自定义设置映射范围中的映射范围的阶数数量。**异常值映射**：您可以单击异常值图形尺寸输入框，自定义设置一个像素数值，用于定义undefined、null、NaN等异常值的映射状态。详情请参见[映射类型](#2d144ba0deuk4)。**自定义映射**：打开开关，您可以自定义输入最大值最小值的图标像素尺寸大小值和该图形尺寸对应的数据值相映射匹配，用于满足您的定制需求。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1379881861/p634007.png)**删除整行映射条目**：单击某一行映射条目左侧的![垃圾桶图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1379881861/p634012.png)图标，即可删除整行映射范围条目。**设置映射字段**：您可以在某一行映射条目右侧的输入框，自定义输入与该范围对应的数据字段值。根据映射范围的**映射类型**不同，您需要输入不同的数值类型的数据字段值内容。**添加映射条目**：单击映射范围列表下方的![添加图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1379881861/p634015.png)图标，即可添加一行范围映射条目。    |

## **映射类型**

范围映射类型分为三大类：**名词分类**型、**数值连续**型和**数值分区**型。

|    **类型**    |    **说明**    |
|    ----    |    ----    |
|    **名词分类**    |    适用于需要给名称和类别等分类型的数据，提供映射范围上的区分。映射上各个映射范围之间互相区分度较大。图 1.名词分类型映射范围模板样式![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7624923371/p634020.png)图 2.名词分类型数据示例图![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2724923371/p882664.png)    |
|    **数值连续**    |    适用于给数值型数据分配连续范围的情况，且数值和范围在映射空间内为一一对应的关系。图 1.数值连续型映射范围模板样式![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2724923371/p882666.png)图 2.数值连续型数据示例图![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2724923371/p882667.png)    |
|    **数值分区**    |    适用于在数值连续型映射的基础上，对连续的数值进行分区，将同一数值区间内的数据归为一类，并将此区间内的数据填充相同范围的情况。使用数值分区型映射，可以减少映射范围数量，避免视觉混乱。图 1.数值分区型映射范围模板样式![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2724923371/p882669.png)图 2.数值分区型数据示例图![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2724923371/p882671.png)    |
