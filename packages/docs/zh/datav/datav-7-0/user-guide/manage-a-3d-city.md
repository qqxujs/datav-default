本文为您介绍如何管理三维城市，包括预览、复制、移动到项目、重命名和删除等功能。

## **预览三维城市**

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，鼠标移动到某个三维城市上。
3. 单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0912919661/p515380.png)图标，进入预览页面。  
![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0912919661/p515442.png)

## **复制三维城市**

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0912919661/p515405.png)图标下的**复制**。  
![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0912919661/p515445.png)
3. 查看复制后的三维城市。

## **移动到项目**

三维城市**移动到项目**功能可以参考数据看板的**移动到项目**功能，具体操作请参见[项目分组管理](https://help.aliyun.com/zh/datav/datav-7-0/user-guide/manage-project-groups#af708e705ful5)。

## **重命名三维城市**

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0912919661/p515405.png)图标下的**重命名**。  
![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0912919661/p515448.png)
3. 输入新名称后，单击**Enter**键，完成重命名操作。

## **删除三维城市**

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0912919661/p515405.png)图标下的**删除**。  
![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0912919661/p515450.png)
3. 在删除弹窗中，再次单击**删除**。

**重要**  
三维城市删除后不可恢复，请谨慎操作。

## **编辑三维城市**

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，鼠标移动到某个三维城市上，单击**编辑**。  
![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0912919661/p515453.png)
3. 三维城市编辑操作请参见[场景编辑器概览](https://help.aliyun.com/zh/datav/datav-7/user-guide/overview-of-scene-editor)。

## **查看三维城市版本历史**

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，选择某个三维城市，单击**查看历史**。  
![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0912919661/p515489.png)
3. 在发布管理页，查看历史版本的信息，包括项目的**版本号**、**发布时间**、**说明**、**状态**和**操作**等信息。  
![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0912919661/p515493.png)

| **参数**   | **说明**                                                                                                                                                                                                                                                                                                                                                                            |
| ---- | ---- |
| **版本号**  | 一个项目可对应多个版本号。最新使用的版本号等于已上线最大版本号。您可以在画布编辑器中使用项目，并在右侧配置面板中单击**更新版本**按键，将项目升级到最新版本后继续使用。                                                                                                                                                                                                                                                                                             |
| **发布时间** | 每一个版本项目的发布详细时间。                                                                                                                                                                                                                                                                                                                                                                   |
| **说明**   | 发布项目时自定义设置的版本情况说明。                                                                                                                                                                                                                                                                                                                                                                |
| **状态**   | 项目版本的上下线状态，包括**已上线**和**已下线**两种。                                                                                                                                                                                                                                                                                                                                                   |
| **操作**   | 针对不同状态下的历史版本的项目，可对项目进行**上线**或**下线**的操作。**上线**：单击一个**未上线**状态下项目版本右侧的**上线**，在弹出的**上线版本**对话框中，单击**确定**即可上线该版本项目内容。![上线区块版本](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3664344261/p282363.jpg)**下线**：单击一个**已上线**状态下项目版本右侧的**下线**，在弹出的**下线版本**对话框中，单击**确定**即可下线该版本项目内容。![下架版本](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3664344261/p259310.jpg) |

**说明**  
默认情况下，一个项目被发布后都处于**已上线**状态。项目上线后可以在数据看板画布编辑器中，单击**更新版本**，将项目升级到最新版本后继续使用。具体使用项目的方法请参见[场景编辑器概览](https://help.aliyun.com/zh/datav/datav-7/user-guide/overview-of-scene-editor)。
4. 单击历史版本面板右上角的![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0912919661/p515506.png)图标，即可关闭历史版本面板。

## **三维城市其他操作**

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，单击**三维城市**模块右侧的**显示全部**。
3. 在全部三维城市页签，通过设置筛选条件或直接搜索，快速寻找和查看三维城市。
