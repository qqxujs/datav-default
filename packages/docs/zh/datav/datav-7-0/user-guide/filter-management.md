 

本文介绍过滤器的一般操作，包括新建、搜索、重命名、编辑和删除等操作。

## **新建过滤器**

1.  登录[DataV控制台](https://datav.aliyun.com/v/console/)。
    
2.  在**功能导航**模块单击过滤器，进入过滤器管理页面。
    
3.  在过滤器卡片页上方，单击**新建过滤器**。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7956322961/p706922.png)
    
4.  在弹出的对话框中输入**过滤器名称**和**CODE**（即代码块）后，单击**创建**。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8956322961/p706923.png)
    

## **搜索过滤器**

1.  登录[DataV控制台](https://datav.aliyun.com/v/console/)。
    
2.  在**功能导航**模块单击过滤器，进入过滤器管理页面。
    
3.  在过滤器卡片页右上角可以做以下操作。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3756871761/p538437.png)
    
    -   单击修改时间，可以按**名称**、**修改时间**和**创建时间**对当前过滤器进行排序。
        
    -   单击搜索框，输入过滤器名称，快速定位所需过滤器。
        

## **重命名过滤器**

1.  登录[DataV控制台](https://datav.aliyun.com/v/console/)。
    
2.  在**功能导航**模块单击过滤器，进入过滤器管理页面。
    
3.  鼠标悬停在需要编辑的过滤器上，单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3756871761/p522237.png)图标。
    
4.  修改后，单击`Enter`键即可。
    

## **编辑过滤器**

1.  登录[DataV控制台](https://datav.aliyun.com/v/console/)。
    
2.  在**功能导航**模块单击过滤器，进入过滤器管理页面。
    
3.  鼠标悬停在需要编辑的过滤器上，单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3756871761/p522244.png)图标。
    
4.  按需修改过滤器的名称及代码。
    
    以MySQL过滤器示例，不同的过滤器内容不同。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3756871761/p538438.png)
    
5.  修改完成后，单击**更新**。
    

## **删除过滤器**

1.  登录[DataV控制台](https://datav.aliyun.com/v/console/)。
    
2.  在**功能导航**模块单击过滤器，进入过滤器管理页面。
    
3.  鼠标悬停在需要删除的过滤器上，单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3756871761/p522247.png)图标。
    

**重要**

过滤器删除后无法恢复，请谨慎操作。
