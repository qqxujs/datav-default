DataV.GeoAtlas是一款线上地理小工具，能进行地理范围选择、地理边界生成和层级生成等功能。本文简述DataV.GeoAtlas工具的基本情况。

DataV.GeoAtlas工具主要由**范围选择器**、**边界生成器**和**层级生成器**三个子工具模块组成，您可以通过打开[DataV.GeoAtlas地理小工具系列](https://datav.aliyun.com/portal/school/atlas/area%5Fselector?spm=a2crr.23498931.0.0.787b15ddCThiIb)，线上体验并使用DataV.GeoAtlas工具的功能。

## 范围选择器

范围选择器是一款数据来自高德，可以自定义快速选择并生成区域范围数据连接的小工具，您可以在范围选择器中，单击选择某个中国地图区域模块后，自动生成多类型数据格式链接以及下载该行政区域完整的边界数据内容，最小粒度到区级，范围选择器功能详情请参见[范围选择器功能介绍](/zh/datav/datav-6-0/user-guide/introduction-to-range-selector-features#concept-2148333)。

## 边界生成器

边界生成器是一款能生产几何面边界数据，支持多种边界数据编辑功能的小工具。如果您熟悉QGIS，即可快速理解边界生成器并上手使用，边界生成器功能详情请参见[边界生成器功能介绍](/zh/datav/datav-6-0/user-guide/introduction-to-features-of-a-boundary-builder#concept-2148336)。

## 层级生成器

层级生成器是一款用于生成自定义下钻区域树形结构数据功能的小工具。层级生成器功能详情请参见[层级生成器功能介绍](/zh/datav/datav-6-0/user-guide/introduction-to-features-of-a-layer-generator#concept-2208160)。
