 

本文介绍空间档案页面中建筑实体基本属性信息。

单击**空间档案**页面中的**建筑信息**，即可打开建筑信息实体详情页面。![建筑信息](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8810342561/p439233.png)

## 建筑信息页面管理

在建筑信息页面中，您可以进行如下功能操作：

**功能**

**操作说明**

**新增建筑信息**

1.  单击页面左上角的**添加**按钮，页面右侧弹出**编辑建筑信息**面板。
    
2.  在页面右侧的面板中添加所需新增的建筑信息，您可以在面板中自定义配置新增建筑的**建筑编码**、**建筑名称**、**标准英文名**、**标准拼音名**、**建筑类型**、**建筑年代**、**楼层**、**关联POI**、**所属行政区**和**详细地址**。
    
3.  配置完成后单击右下角的**确定**即可完成添加建筑信息；单击**取消**即可取消添加新建筑信息。![添加建筑信息](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2882270261/p243415.png)
    

**查看建筑信息**

在建筑信息页面中，所有新添加的或者编辑过后的建筑信息都会显示在页面中间列表中。![建筑信息列表](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2882270261/p243413.jpg)

**搜索建筑信息**

当您需要查看一个建筑信息且建筑信息在页面中数量过多时，您可以单击页面上方的**搜索**处，自定义输入建筑名称后，单击右侧的![搜索框](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5110270261/p243433.jpg)即可查找到所需的建筑信息。![搜索建筑信息](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2882270261/p243434.png)

**编辑建筑信息**

当您需要修改页面中已有的建筑信息内容，在待修改的建筑信息**操作**列单击**编辑**，在弹出的**编辑建筑信息**面板内修改详细建筑信息后，单击下方**确定**即可。

**删除建筑信息**

您可以在页面中，在待删除的建筑信息**操作**列单击**删除**，即可删除当前建筑信息内容。
