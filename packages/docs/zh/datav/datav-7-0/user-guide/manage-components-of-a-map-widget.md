本文介绍地图子组件管理面板的操作步骤和基本功能，方便您更加快速准确地使用地图组件。

## 操作步骤

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，选择任一数据看板，单击**编辑**，进入画布编辑器页面。
3. 单击左侧**组件库**页签，选择**地图**下的任一组件，单击添加到画布中。
4. 以**基础平面地图**组件为例。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2245501961/p701873.png)
5. 在画布中单击**基础平面地图**组件，在右侧配置栏单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5174632761/p540573.png)图标，展开子组件。同时会开启**子组件面板**和**子组件管理面板**。也可以单击子组件管理面板的![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5174632761/p540610.png)图标，打开子组件面板。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2995666071/p764156.png)  
在子组件页面，您可以单击任一子组件添加到子组件；在子组件管理页面，您可以完成[添加子组件](#section-hn6-rui-toa)、[按颜色归类子组件](#section-vis-tn2-u0j)、[控制子组件显隐](#section-rja-vqm-8uv)、[复制子组件](#section-zac-27w-mjv)、[重命名子组件](#41c0fed087y23)、[删除子组件](#d769ca3087he4)等管理功能。

## 添加子组件

在子组件页面，单击任一子组件缩略图，即可完成子组件的添加。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5174632761/p540581.png)

## 按颜色归类子组件

1. 在**子组件管理**页面，选择同一层级下的任意一个或多个子组件。
2. 单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5174632761/p540584.png)图标，为当前选中子组件设置标签颜色。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5174632761/p540585.png)
3. 单击图标，根据标签颜色筛选出对应颜色标记类别的子组件。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5174632761/p540589.png)

## 控制子组件显隐

移动鼠标至**子组件管理**列表中的任意一个子组件上，单击右侧的![隐藏](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4622637851/p99437.png)图标，即可在地图中显示或者隐藏当前选中的子组件。您也可以单击右键选择**隐藏**，隐藏当前子组件。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5174632761/p540590.png)

## 复制子组件ID

移动鼠标至**子组件管理**列表中的任意一个子组件上，单击右键选择**复制ID**，即可复制当前选中的子组件ID。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2245501961/p701882.png)

## 复制子组件

移动鼠标至**子组件管理**列表中的任意一个子组件上，单击右键选择**复制**，即可复制当前选中的子组件。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2245501961/p701883.png)

## 删除子组件

**重要** 

子组件删除后将无法恢复，请谨慎操作。

移动鼠标至**子组件管理**列表中的任意一个子组件上，单击右键选择**删除**，即可删除当前选中的子组件。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2245501961/p701885.png)

## 重命名子组件

移动鼠标至**子组件管理**列表中的任意一个子组件上，双击子组件名称，在子组件名称输入框中输入文字，即可重命名当前选中的子组件。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5174632761/p540605.png)

## 排序子组件

移动鼠标至**子组件管理**列表中的任意一个子组件上，上下拖动即可调整子组件在所属分层区域内的层级位置。

子组件所属分层分为三大块，分别为**置顶层**、**中间层**和**底部层**。每一层级区域内包括如下子组件。 

* **置顶层**：轮播标签层。
* **中间层**：点热力层、背景图片层、网格热力层、飞线层、蜂窝热力层、等值面层、信息标签层、中国国家边界、轨迹线层、自定义区域下钻层、矢量散点层、呼吸气泡层、图标散点层、符号聚合层、区域热力层。
* **底部层**：基础底图层和高德底图层。

**说明** 

只能在子组件所属的区域块内移动调整层级关系，不可跨区域调整子组件层级。
