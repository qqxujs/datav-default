本文介绍设计库的概念、资产分类和如何使用设计库制作可视化大屏。

**说明** 

设计库功能为7.0版本产品新增功能，请根据[新版控制台](https://datav.aliyun.com/v/console/)环境进行操作。

## **什么是设计库**

DataV基于用户搭建可视化大屏的标准链路，帮助用户快速检索可视化大屏所需要的模块单元，方便用户通过拖拉拽的方式，在标准的设计规范下，做出高质量设计水平的可视化大屏。那设计库和组件库有什么区别呢？组件通常包含了一套默认样式，难以同时满足各类使用场景。在设计库中，每一个标题、面板、地图都经过了设计师精心配置和优化，风格任意挑选、开箱即用、自由组合，大大提升了大屏搭建效率。

## **可视化套件类型**

由于用户在设计过程中，常常在创建项目时会使用模板创建，但是单个模板拥有的数据面板或者地图，往往不能满足用户多元化的场景需求。当使用其他模板中的素材时，又会遇到风格不匹配的问题，导致页面的拼凑感严重，影响美观。DataV在设计库中推出5款可视化套件：**晴山蓝**、**丹霞橙**、**暮山紫**、**碧玺绿**和**云峰白**。您可以单击页面左上角的**全部**下拉菜单选择需要使用的可视化套件。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5281669861/p694763.png)

## **案例演示**

本案例介绍通过使用可视化套件如何快速搭建数据大屏。

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，将鼠标移动到**创建PC端看板**上，单击**选择模板**。
3. 在**创建看板**页面，选择**空白看板**，单击**创建看板**。
4. 输入看板名称后，单击**创建看板**。
5. 在左侧导航栏单击**设计库**，进入设计库页面。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4538749761/p608286.png)
6. 单击设计库页面左上角的**全部**，在下拉列表中选择**碧玺绿**。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4538749761/p608296.png)
7. 单击**设计规范**，选择**主视觉居中布局**。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4538749761/p608300.png)
8. 单击**地图**，在主地图区域插入**3D省份地图**。拖拽编辑地图大小镶嵌到主地图区域。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4538749761/p608303.png)
9. 单击**标题**，在背景块添加标题组件。查看自定义背景块的尺寸大小，将标题组件契合在背景块上方。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4538749761/p608318.png)
10. 单击**数据面板**，选择需要添加的资产。分别调整每个资产的尺寸和样式，贴合在自定义背景块上方。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4538749761/p608344.png)
11. （可选）在素材中选择需要添加的资产，美化完善数字大屏。
12. 完成设计后，单击画布编辑器页面右上方的预览，预览数字大屏。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4538749761/p608350.png)
