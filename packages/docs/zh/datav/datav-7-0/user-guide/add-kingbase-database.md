在DataV-Board（数据看板）中使用人大金仓数据库之前，需要先将人大金仓数据库实例添加为DataV-Board的数据源。本文为您介绍如何添加人大金仓数据库。

## 前提条件

已准备好待添加的人大金仓数据库实例信息。

## 使用限制

仅支持DataV企业版及以上版本。

## 通过内网添加数据源

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，单击**数据准备** \> **数据源** \> **新建数据源**。
3. 从**类型**列表中，选择**人大金仓**。
4. 在列表中选择**内网**，并选择人大金仓数据库实例所在的地域。
5. 填写数据库配置信息。

| **参数**       | **说明**                                    |
| ------------ | ----------------------------------------- |
| **VPC ID**   | VPC实例的ID，此VPC为人大金仓数据库实例所在的VPC。            |
| **VPC 实例ID** | 可选VPC数据库实例 ID、 VPC 网络连接地址、VPC 网络IP地址其中一种。 |
| **名称**       | 自定义的数据源名称。                                |
| **域名**       | 访问数据库实例使用的VPC地址。                          |
| **端口**       | 数据库设置的端口号。                                |
| **用户名**      | 登录数据库的用户名称。                               |
| **密码**       | 登录数据库的密码。                                 |
| **数据库**      | 自定义输入数据库名称。                               |
6. 数据库信息填写完成后，单击**确定**，系统将自动测试连接。连接成功后，新添加的数据源将显示在数据源列表中。

## 通过外网添加数据源

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，单击**数据准备** \> **数据源** \> **新建数据源**。
3. 从**类型**列表中，选择**人大金仓**。
4. 在列表中选择**外网**。
5. 填写数据库配置信息。

| **参数**  | **说明**                |
| ------- | --------------------- |
| **名称**  | 自定义的数据源名称。            |
| **域名**  | 访问人大金仓数据库实例使用的外网地址。   |
| **端口**  | 数据库设置的端口号。            |
| **用户名** | 登录数据库的用户名。            |
| **密码**  | 登录数据库的密码。             |
| **数据库** | 自动获取数据库列表或自定义输入数据库名称。 |
6. 数据库信息填写完成后，单击**确定**，系统将自动测试连接。连接成功后，新添加的数据源将显示在数据源列表中。

## **后续步骤**

数据源创建完成后，您可按需执行如下操作：

* 使用数据源：您可进入目标项目的编辑页面，添加组件并使用数据源。
* 管理数据源：根据需要执行数据源的编辑、删除等管理操作，详情请参见[数据源管理](https://help.aliyun.com/zh/datav/datav-7-0/user-guide/manage-data-sources-1)。
