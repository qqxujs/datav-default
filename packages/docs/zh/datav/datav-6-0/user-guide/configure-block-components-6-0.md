本文介绍区块内组件的样式查看及配置的方法，帮助您理解区块的样式功能。

## **前提条件**

已经完成[区块创建](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/manage-a-block#7f2a4c14feoal)。

## 操作步骤

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，单击**组件开发**模块下的**区块**，进入区块管理页面。
3. 单击目标区块，进入该区块的编辑器页面中。
4. 在区块编辑器中，添加组成区块所需的组件。
5. 单击已添加的某个组件，即可在编辑器右侧查看到组件的所有样式配置内容，如下图所示。  
![区块组件样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3090903461/p283510.png)

**说明**  
您可以参见画布编辑器中[配置组件样式](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/configure-item-description#concept-lsz-bc4-t2b)的方法，对区块内的组件样式进行自定义调整。
6. 单击区块编辑器页面右上角的![预览区块图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4123054261/p254797.jpg)图标预览区块。
7. 在预览页面，单击样式面板，可以查看区块内组件的所有样式内容，并可以在样式面板内查看[样式](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/description-of-configuration-items)、[数据](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/configure-a-data-interface)、[事件](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/configure-an-event-interface)和[动作](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/configure-an-action-interface)的配置详情。![查看全量组件样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4090903461/p284179.png)

**说明**  
目前区块内添加的组件暂时不支持组件样式的重组功能，在样式面板中会全量展示区块内部所有组件的样式。
