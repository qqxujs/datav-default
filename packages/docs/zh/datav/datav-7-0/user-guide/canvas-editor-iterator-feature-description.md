迭代器即列表生成器，列表中每一项称为一个迭代单元，单元中的组件类型及布局与单元模板相同。单元中组件的数据可由自身数据源驱动，或者从迭代器中获取。

**重要** 

迭代器的一级子图层只能是迭代单元，不能是普通组件，普通组件包含在迭代单元内。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6546482671/p1025677.png)

## 迭代器功能操作

| **操作**    | **说明**                                                                                                                                                                                                                                                                                |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 生成迭代器     | 在画布编辑器面板中，单击**组件列表**中的一个或多个组件，添加组件到画布中，您可以将添加的组件成组。单选或多选组件和组，选中后右键单击，在弹窗中选择**生成迭代器**，被选中的组件会自动包裹在迭代器内部，生成一个迭代单元。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6546482671/p1025679.png)                                                                        |
| 向迭代器中新增组件 | 在画布编辑器图层面板中，单击选中某个不包裹在迭代器内的组件，拖动该组件图层移动到迭代器中，即可向迭代器中新增一个组件。                                                                                                                                                                                                                           |
| 从迭代器中删除组件 | 在画布编辑器图层面板中，单击选中某个包裹在迭代器内的组件，拖动该组件移动到迭代器外的图层中，即可从迭代器中删除一个组件。                                                                                                                                                                                                                          |
| 从迭代器获取数据  | 迭代器数据响应结果应为复杂数组类型，数组的每一项数据可向内传递给子级组件。根据就近原则，迭代器内组件可获取最近一级父级迭代器的单元数据项，经过组件过滤器，得到最终的响应结果。组件从迭代器中获取数据操作步骤如下：选中迭代器中的某个组件，打开右侧组件**数据源配置**面板。在**数据源配置**面板中，单击**数据源类型**下拉框，选择**从迭代器中获取**数据源选项。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6546482671/p1025686.png) |
| 重命名迭代器    | 在画布编辑器图层栏，您可以双击迭代器，重命名输入迭代器的名称。                                                                                                                                                                                                                                                       |
