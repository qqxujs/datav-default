 

项目分组管理功能可以对已创建的数据看板进行归类，方便用户进行管理和使用。本文介绍如何使用DataV的项目分组管理功能。

## **操作步骤**

1.  登录[DataV控制台](https://datav.aliyun.com/v/console/)。
    
2.  在**工作台**页面，单击**我的项目分组 > 新建项目分组**。
    
    ![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2456419861/p692904.png)
3.  在分组名称编辑框内，输入分组名，完成后单击空白处或者按**Enter**键确定。
    
    您可以单击![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1456419861/p692908.png)图标或![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1456419861/p692909.png)图标对已经生成的分组进行**重命名**或**删除**操作。
    
    **说明**
    
    删除分组后，该分组中的数据看板将全部回归到**未分组**。
    
4.  在数据看板中，单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9472919661/p514899.png)下的**移动到项目分组**，将看板移动到左侧新建的分组栏中。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8633028961/p730895.png)
    
5.  在弹窗中选择目标项目分组，单击**确定**。
    
    ![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1456419861/p692914.png)
6.  在新建的分组中查看移动后的结果。
    
7.  （可选）单击**回收站**，您可以查看**24小时内**的已删除的数据看板和三维城市内容。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2456419861/p692918.png)
    
8.  （可选）鼠标hover在需要还原的数据看板或三维城市上，您可以单击**还原**，将当前内容还原至删除前的分组中。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1456419861/p692919.png)
