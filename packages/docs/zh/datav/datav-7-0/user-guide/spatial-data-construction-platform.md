空间数据构建平台是一个专用于城市级三维场景创建、管理和可视化的专业工具，空间数据管理页面内提供了我的主页、我的项目、我的数据和空间档案四大功能模块，用户可以在这里创建项目，上传数据和添加模型等。本文为您介绍空间数据管理页面的结构与功能，帮助您快速使用空间构建平台。

## **使用限制**

空间数据构建平台仅支持**DataV专业版**及以上版本使用。

## 进入空间构建平台

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，鼠标单击**创建三维城市**，在弹框中输入名称后，单击**确定**。
3. 在**选择创建方式**对话框内选择**空间构建数据**模式并单击**创建**。
4. 在**选择空间数据项目**对话框内，单击右上角的**前往空间构建平台创建项目**，即可进入空间构建平台。  
![空间构建平台入口](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2124695161/p240473.jpg)

## 使用空间构建平台

1. 在**空间数据管理**页面，单击**我的项目**，进入我的项目页面。
2. 单击页面上方的**创建项目**，输入**项目名称**后，在**添加基底数据**对话框中配置数据。  
![创建项目](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2317612561/p437722.png)

  1. 在**添加基底数据**对话框内，您可以在左侧**空间实体**列表中选择基底内容。
  2. 在**添加基底数据**对话框右侧的地图区域处，您可以通过选择地图版本和城市快速定位目标区域。使用鼠标滚轮自由缩放视图，或通过框选工具精确划定需要展示的[区域范围](https://help.aliyun.com/zh/datav/datav-7-0/user-guide/add-project-base-data#section-5ll-l20-0gw)。
  3. 单击对话框右下角**创建项目**，进入项目编辑器页面。  
  ![添加基底数据](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1317612561/p437728.png)
3. 在[项目编辑器页面](https://help.aliyun.com/zh/datav/datav-7-0/user-guide/manage-a-project-editor-page)中，您可以灵活管理实体空间的基底数据和项目通用元素，并可通过右侧地图绘制面板对任一数据进行可视化编辑与自定义调整。

**说明**  
系统默认定位为杭州区域，您需要手动调整到指定区域。
4. 内容修改完成后，单击页面左上角的**返回总览**，切换到**我的项目**页面。
5. 在**空间数据管理**页面，单击左侧列表中的**我的数据**，可以查看页面内的数据分组状况以及所有添加完成的数据内容，并进行上传和管理数据操作。  
![我的数据](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1872522561/p437953.png)
6. 单击左侧的**空间档案**，进入**空间档案**页面，您可以浏览各类建筑模型信息，并可以调用系统预制模型或上传自定义模型进行使用。  
![空间档案](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1872522561/p437958.png)
