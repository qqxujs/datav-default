设计资产在可视化应用搭建的场景下对于提升视觉效果十分重要，设计资产管理的功能可以帮助用户更好地管理自己的设计资产，实现资产的分类、复用和分享，并提升可视化应用设计的效率与效果。本文介绍设计资产的概念及使用功能。

## 概念说明

设计资产指的是用以增强可视化应用视觉效果的设计元素，包括图片、视频和模型等。 

**说明** 专业版及以上版本用户，可管理及使用模型类的设计元素。 

## 设计资产使用界面

可视化应用画布中添加了可关联使用设计资产内容的组件，并在您更改组件所使用的设计资产时，会弹出设计资产的对话框。您可以在对话框中使用上传、检索、使用资产等功能操作，详情请参见[使用界面功能说明](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/use-interface-function-description#concept-2094777 "当可视化应用中添加了使用设计资产功能的相关资产，在需要更新其图片、视频和模型的配置项时，您可以快速切换使用官方或自己的设计资产，搜索资产及排序资产等，本文介绍设计资产使用界面中各个功能的详细说明。")。![使用界面面板](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4742776261/p294176.png)

目前，支持使用设计资产功能的组件及样式配置如下表格所示。 

| 组件资产    | 版本           | 配置             | 设计资产类型 |
| ---- | ---- | ---- | ---- |
| 所有组件    | 任意版本         | 组件中上传图片控件的配置项。 | 图片     |
| 视频组件    | V2.5.10版本及以上 | 视频地址获取的配置项。    | 视频     |
| 模型查看器组件 | 任意版本         | 模型地址获取的配置项     | 模型     |

## 常规方法与设计资产功能方法对比情况

* 常规方法：通常使用的是图片或者视频类资产，您可以通过在组件右侧配置面板中，上传本地资产或粘贴资产URL链接的方式上传所需资产。  

**注意** 这种资产使用方式需要您在本地进行管理可视化应用所需要的所有资产。您在使用时，需要重新上传或者贴入地址等操作。一方面对本地电脑空间造成了很大冗余；另外一方面，本地资产也没有同步功能，在您更换工作环境（电脑等）后，就无法再次使用。
* 设计资产功能方法：您在使用组件右侧配置面板中关联了设计资产功能的配置项时，可直接在设计资产对话框中快速使用所需的设计资产内容，具体使用设计资产功能的方法请参见[素材或模型调用设计资产操作示例](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/material-or-model-call-design-asset-action-example#task-2094926 "为了提升可视化应用设计的效率与效果，DataV在资产库面板中新增了素材和模型两大分类，为您提供了大量便捷优质的设计素材物料。本文介绍如何在DataV画布编辑器中使用素材或模型，并通过关联的更改设计资产功能调用设计资产内容。")。  

**说明** 这种资产使用方式无需将所需资产存放在本地工作环境，您可以将您现有的资产上传到资产管理页面内，在您后续使用中将会跟随您的账号在任何工作环境内直接调用，提高了资产使用的效率及便捷程度。
