## 配置

![蜂窝热力层配置面板v2.x](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2468068951/p82331.jpg)

-   搜索配置：单击配置面板右上角的搜索配置，可在搜索配置面板中输入您需要搜索的配置项名称，快速定位到该配置项，系统支持模糊匹配。详情请参见[搜索资产配置](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/search-for-configurations#task-2376842 "通过搜索资产配置功能，您可以快速定位到资产的对应配置项，在搜索配置页面直接修改配置样式，也可以复用同类型的配置样式。")。
-   可见级别：设置蜂窝热力层的可见级别值，在设定的级别范围内蜂窝热力层可见，级别之外不可见。
-   蜂窝样式：设置蜂窝大小、透明度和填充颜色等参数。
    
     
    
    参数
    
    说明
    
    蜂窝大小
    
    蜂窝热力层的蜂窝大小。DataV会根据您设置的蜂窝大小和数据范围计算出所有蜂窝的数量和位置。
    
    透明度
    
    设置蜂窝热力层的蜂窝透明度值。
    
    填充颜色
    
    蜂窝热力层蜂窝填充颜色包括最小值、最大值和颜色分级。
    
    -   最小值：蜂窝数最少点的颜色，请参见[颜色选择器说明](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/configure-item-description#section-kdw-vj4-t2b)进行修改。DataV会根据数据中点的经度纬度值，计算出点所在的蜂窝，其中点个数最少的蜂窝就为最小值蜂窝。
    -   最大值：蜂窝数最多点的颜色，请参见[颜色选择器说明](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/configure-item-description#section-kdw-vj4-t2b)进行修改。DataV会根据数据中点的经度纬度值，计算出点所在的蜂窝，其中点个数最多的蜂窝就为最大值蜂窝。
    -   颜色分级：蜂窝热力层的颜色分级数。与最小值和最大值配置项配合使用，根据点的分布，设置每个蜂窝的颜色。
    
    **说明** 在最大值蜂窝和最小值蜂窝之间的蜂窝，会根据您设置的颜色分级数，对应显示最大值和最小值之间的颜色。
    
    阴影
    
    设置蜂窝阴影的颜色和阴影模糊的程度范围。
    
    边线样式
    
    蜂窝边框线的颜色和粗细，单位为px。
    
-   标注：蜂窝的文字注释。单击标注左侧的![眼睛图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8648413261/p89095.jpg)图标，可控制标注的显隐。
    
    文本样式：设置标注文字的字体、文字粗细、字号和颜色。请参见[颜色选择器说明](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/configure-item-description#section-kdw-vj4-t2b)进行修改颜色。
