巡航标记层是三维城市地图的子组件，支持独立的样式和数据配置，包括巡航标记层的颜色、色彩混合模式、缩放倍数和移动速度等参数。巡航标记层子组件以表示人或物体的标记在指定的轨迹上移动巡航。本文介绍巡航标配置项的含义。

## 配置

![巡航标记配置面板](http://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4438068951/p147524.png) 

巡航标记配置面板参数说明。 

| 参数   | 说明                             |
| ---- | ---- |
| 颜色   | 巡航标记的颜色。                       |
| 色彩混合 | 巡航标记色彩混合的模式，可选：正常模式或叠加模式。      |
| 缩放倍数 | 设置巡航标记的缩放倍数，数值越大，巡航标记被放大的体积越大。 |
| 圆点高度 | 设置巡航标记上方圆点距离地平面的高度，取值范围1\~200。 |
| 移动速度 | 设置巡航标记的移动速度。                   |

## 数据

![巡航标记路径数据面板](http://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5438068951/p147755.png) 

__表 1\. 字段说明__
| 字段 | 说明                   |
| ---- | ---- |
| x  | 巡航标记中心点所经过的路径点所处的经度。 |
| y  | 巡航标记中心点所经过的路径点所处的维度。 |
| z  | 巡航标记中心点距离地表面的高度值。    |

## 交互

此组件没有交互事件。

## 蓝图编辑器交互配置

1. 在画布编辑器页面，单击三维城市子组件管理列表中巡航标记右侧的![导出到蓝图编辑器](http://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5438068951/p89089.jpg)图标。
2. 单击页面左上角的![蓝图图标](http://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5438068951/p89087.jpg)图标。
3. 在蓝图编辑器配置页面，单击导入节点栏内的巡航标记组件，在画布中可以看到如下图所示的巡航标记的蓝图编辑器配置参数。![巡航标记蓝图参数](http://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5438068951/p147758.png)  
  * 事件  
  当巡航路径请求完成时：数据接口请求返回并经过过滤器处理后抛出的事件，同时抛出处理后的JSON格式的数据。具体数据请参见上文的[数据示例](#section-emm-elk-oxi)。
  * 动作  

| 动作     | 说明                                                                                                                       |
| ---- | ---- |
| 请求巡航路径 | 重新请求服务端数据，上游数据处理节点或图层节点抛出的数据将作为参数。例如巡航标记配置了API数据源为http://api.test，传到请求巡航路径动作的数据为{ id: '1'}，则最终请求接口为http://api.test?id=1。 |
| 导入巡航路径 | 按组件绘制格式处理数据后，导入组件，重新绘制。不需要重新请求服务端数据。具体数据请参见上文的[数据示例](#section-emm-elk-oxi)。                                              |
| 显示组件   | 显示组件，不需要参数。                                                                                                              |
| 隐藏组件   | 隐藏组件，不需要参数。                                                                                                              |
