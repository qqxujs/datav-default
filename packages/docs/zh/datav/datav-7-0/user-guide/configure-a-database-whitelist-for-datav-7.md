## 11网段网络不通问题处理

如果遇到11网段网络不通的问题，请执行如下命令，在_/etc/rc.local_中添加路由。 

```
sudo route add -net 11.0.0.0/8 gw 10.152.28.247
```

**说明** 

_gw_后面的IP地址为您服务器本机的网关。

如果问题仍无法解决，请提交工单到ECS，告知阿里云技术支持工程师，11网段的服务器无法访问到您的服务器。
