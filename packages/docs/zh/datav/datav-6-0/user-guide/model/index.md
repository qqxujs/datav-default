本文介绍模型类模块内组件的结构及说明。

## **模型组件结构说明**

* 在模型模块内的二级组件面板中，有**建筑**、**工业**和**产品**三种模型类型组件。
* 所有的模型，其组件类型只有**模型查看器**一种，组件的详细配置说明请参见[模型查看器](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/model-viewer-1#concept-2156552)。
* 您可以在画布中添加模型组件内容，按照**模型查看器**组件配置的使用方法，来自定义修改，包括可配置模型的位置移动、大小缩放、样式配置、数据配置和蓝图配置等，详情请参见[组件指南](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/configure-item-description#concept-lsz-bc4-t2b)和[资产管理](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/asset-list-management#concept-2088328)。
* 模型组件可在更改关联设计资产功能的配置项时，调用设计资产内容，详情请参见[素材或模型调用设计资产操作示例](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/material-or-model-call-design-asset-action-example#task-2094926)。
