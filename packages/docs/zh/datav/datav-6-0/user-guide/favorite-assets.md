  收藏使用和管理资产-DataV数据可视化-阿里云

您可以通过收藏资产功能，收藏可视化应用中的任意资产，方便后期有需要时，随时调用收藏栏中的资产。

## 前提条件

确保DataV版本为企业版及以上版本。

**注意** DataV对收藏资产的个数有限制，企业版支持最多收藏20个资产，专业版支持最多收藏40个资产，尊享版支持最多收藏1000个资产。

## 收藏单个资产

1.  登录[DataV控制台](https://datav.aliyun.com/)。
2.  在我的可视化页面，单击您创建的可视化应用项目上方的编辑按钮。
3.  在画布编辑器页面，执行以下步骤收藏资产。
    
    **注意** 在收藏资产之前，请确保您的可视化应用项目中已经添加了相应的资产，否则需要首先添加资产，详情请参见[添加资产](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/add-a-widget-1/#task-2375434 "通过添加资产功能，您可以在DataV可视化应用项目中添加可视化资产，配置资产所需要的应用样式。本文介绍在画布编辑器中添加资产的方法。")。
    
    1.  在图层栏或画布区中单击选中一个资产。
    2.  右键单击选择收藏。
        
        **说明** 收藏资产的数据结果会受到数据源变更的影响，建议在收藏时使用静态数据。
        
    3.  在收藏对话框中，输入收藏资产的名称（默认为资产列表中的显示名称）。
    4.  单击确定。
4.  资产收藏成功后，在全部资产页面中，单击左侧导航栏的收藏图标，查看您收藏的资产。
    
    ![收藏组件](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3164447261/p267250.png)
    
5.  单击您收藏的资产，可将该资产应用到可视化应用项目中。

## 收藏多个资产

您可以使用分组收藏资产功能，将多个资产进行归类收藏，形成一个资产合集，方便查找使用。

由于DataV对收藏资产的数量有限制，通过分组收藏资产功能，您还可以节省资产收藏的配额数量。

**注意** 分组收藏资产功能为专业版及以上版本功能，如果您需要使用此功能，请将先DataV升级到专业版或尊享版。

1.  在图层栏或画布区中，按住Ctrl键，单击选中多个资产。
2.  右键单击选择成组，成组多个资产。
3.  选中已成组的资产。
4.  右键单击选择收藏，可将多个已成组资产一起收藏至资产收藏夹中。
    
    **注意** 收藏的资产数据源如有变更，可能导致资产不可用，建议在收藏时资产使用静态数据。
    

## 查看收藏的资产

资产收藏成功后，您可以执行以下步骤，查看收藏的资产。

1.  返回到控制台首页。
2.  单击我的资产，单击页面左侧的资产收藏夹。
3.  在资产管理页面，查看您已经收藏的资产以及还可以收藏的资产数量。
    
    ![查看收藏组件](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1805340561/p431788.png)
    

## 删除收藏的资产

将鼠标移至收藏的资产上，单击删除图标即可将该资产从收藏夹删除。![删除收藏组件](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4170858951/p54019.png)

## 重命名收藏的资产

将鼠标移至该资产名称处，单击重命名图标输入文本即可修改资产名称。

![重命名收藏组件](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4170858951/p54024.gif)

**说明** 您也可以在收藏资产时，修改资产名称。

## 升级收藏资产为区块

将鼠标移至收藏的资产上，单击升级成区块图标即可选择升级区块，详情请参见[升级已收藏的资产为区块](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/upgrade-favorited-assets-to-blocks#task-2086230 "本文介绍如何将收藏的历史资产升级成区块。")。
