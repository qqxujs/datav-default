本文介绍如何使用三维视图与相机编辑面板内相机视角调整与编辑功能。

## 调整三维视图相机视角

![三维视图内快捷键提示及数据反馈提示](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5813268461/p421029.png)

三维视图中，可使用多种相机视角控制功能，例如支持平移、旋转和缩放等操作，您可以通过鼠标调整三维相机视角画面，详细操作方法如下所示：

* 单击**alt**键，并按住鼠标左键拖拽，可以旋转相机视角。
* 单击并按住鼠标左键拖拽，可以拖动平移相机视角。
* 上下滚动鼠标滑轮或使用触控板缩放，可以缩放相机视角。

**说明** 

如果您对当前三维视图中的基底数据内容不满意，可以单击三维视图内的反馈超链接，跳转到反馈页面，提出您的宝贵意见。

## 相机路径面板与相机高程面板功能说明

多视图相机编辑器，支持以第三人称视角在经纬度二维空间和高程一维空间随心所欲编辑相机的空间路径和姿态，其中相机经纬度路径支持线性插值模式和贝塞尔插值模式，方便您编辑出想要的任意曲线。关键帧面板中的相机属性、三维视图、路径面板和高程面板，四个面板在多视图相机编辑过程中实现了联动响应，详细功能说明如下所示。

**说明** 

在三维视图和相机编辑器面板中，可以编辑当前分镜下某个关键帧的相机属性。

|    **功能**    |    **说明**    |
|    ----    |    ----    |
|    编辑某一帧相机经纬度    |    ![更改经纬度](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3562268461/p416790.png)功能操作说明：在关键帧面板内，单击选中某一帧。在相机路径面板中，拖动面板中的相机图标，相机图标位置更改后，即可改变该选中帧的相机经纬度。**说明** 场景编辑器内的相机路径面板，默认选择使用**线性插值**编辑页面。单击下方关键帧面板左侧的**添加关键帧**图标，即可保存当前帧的相机属性，并为当前场景新增一个关键帧。    |
|    更改已有关键帧的相机经纬度    |    ![更改关键帧锚点位置](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3562268461/p416799.png)功能操作说明：在关键帧面板内，单击选中某一已有的关键帧。在相机路径面板中，拖动面板中的关键帧锚点，完成锚点位置更改后，即可更新当前关键帧的相机经纬度。    |
|    更改已有关键帧的相机高程    |    ![更改高程距离](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3562268461/p416808.png)功能操作说明：在关键帧面板内，单击选中某一已有的关键帧。在相机高程面板中，上下拖动高程面板中的相机图标，完成图标高度位置更改后，即可更新当前关键帧的高程数据。    |
|    使用贝塞尔路径插值    |    ![修改贝塞尔插值](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3562268461/p416829.png)功能操作说明：在关键帧面板内，单击选中某一已有的关键帧。在相机路径面板中，单击面板左侧的![贝塞尔图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3562268461/p416813.jpg)图标，切换贝塞尔插值编辑。在相机路径面板中，拖动面板中的当前关键帧两端的贝塞尔控制点，完成贝塞尔控制点位置更改后，即可更新当前关键帧的贝塞尔插值。    |
|    查看相机完整路径    |    单击相机路径面板左侧的![查看完整路径](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3562268461/p416846.jpg)图标，即可定位并在相机路径面板内居中展示当前分镜下多个关键帧的完整路径。    |
|    缩放相机路径面板    |    单击相机路径面板左侧的![加号](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2562268461/p416848.jpg)和![减号](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2562268461/p416849.jpg)图标，即可放大或缩小相机路径面板的地图大小。**说明** 您也可以将鼠标移动到相机路径面板上，上下滚动鼠标滑轮或使用触控板缩放，控制相机路径面板内的地图大小缩放。    |
