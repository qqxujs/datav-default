  发布三维城市构建器项目后在画布中使用-DataV数据可视化-阿里云

通过三维城市构建器项目的发布功能，可以将已经开发完成的三维城市构建器项目发布到线上环境，后续在资产面板直接查看并消费使用。本文介绍如何发布一个三维城市构建器项目以及如何在画布编辑器中使用该项目。

## 发布项目

1.  登录[DataV控制台](https://datav.aliyun.com/)。
2.  在三维城市构建器页面中，选择一个三维城市构建器项目，单击编辑。
3.  在项目场景编辑器中，单击页面工具栏右上角的![发布图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1343458461/p422384.jpg)图标。
4.  在弹窗的发布对话框中，输入本次项目发布的备注说明内容，再单击发布。![发布项目](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0262268461/p413824.jpg)
    
    **说明** 发布后的项目可以在DataV画布编辑器中的全部资产 > 城市中找到并被添加使用，您也可以参见[编辑项目](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/edit-projects#task-2194509 "本文介绍编辑三维城市构建器项目的方法，帮助您快速修改三维城市构建器项目的内容和配置。")，重新编辑修改项目内容。
    

## 查看并添加使用项目

发布后的三维城市构建器项目可直接在画布中被调用使用，具体步骤如下：

1.  登录[DataV控制台](https://datav.aliyun.com/)。
2.  在我的可视化页面中，选择一个可视化应用，单击编辑。
3.  在画布编辑器页面，单击左上角的![组件列表](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8451419161/p267288.png)图标，展开资产列表。
    
    在资产列表区域中，默认显示全部资产模块内容。
    
4.  在全部资产列表下，单击城市模块，即可查看到已发布的所有三维城市构建器项目内容。![选择已发布城市项目](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1343458461/p413855.png)
5.  单击某个三维城市构建器项目，即可添加该项目到右侧画布面板中，您可以参见组件在画布编辑器内的使用方式，在画布中调整三维城市构建器项目的大小、位置和交互等功能，还可以导出该项目到蓝图编辑器，可以使用之前在项目场景编辑器内，自定义配置的区块接口内容，配置项目的蓝图交互效果，详情请参见[蓝图编辑器配置说明](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/blueprint-editor-configuration#concept-2193562 "三维城市构建器项目内的蓝图编辑器功能，用于编辑项目黑盒产物中的逻辑控制。本文介绍三维城市构建器项目中蓝图编辑器的使用方法。")。
