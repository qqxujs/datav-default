 

本文介绍如何使用模板创建移动端数据看板。

## **操作步骤**

1.  登录[DataV控制台](https://datav.aliyun.com/v/console/)。
    
2.  在**工作台**页面，移动鼠标到**创建移动端看板**上，单击**选择模板**。
    
    ![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7366519861/p693109.png)
3.  在**创建看板**页面，通过**应用场景**、**设计风格**和**颜色**三种维度筛选出自己需要的模板后，单击**创建看板**。
    
    ![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7366519861/p693124.png)
4.  在**创建移动端看板**弹窗中，输入**看板名称**，（可选）选择**项目分组**后，单击**创建看板**。
    
    创建成功后，系统会跳转到数据看板编辑页面。
    
    ![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1179972761/p526429.png)

**说明**

您也可以在**设计中心 > 移动端**，选择**空白看板**或系统预置模板创建数据看板。
