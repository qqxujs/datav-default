DataV-Board 7.0（数据看板）由可视化工作台、设计中心、可视化学院、帮助和教程以及可视化应用中心组成。本文简要介绍了 DataV-Board控制台的各模块组成，让您快速了解界面布局。

## **可视化工作台**

### **工作台**

工作台是您创建数据看板的核心入口，在工作台您可以进行数据准备、组件开发、资源管理、创建数据看板和查看数据看板等操作。![1.jpg](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4571419861/p692795.jpg)

|    **序号**    |    **区域名称**    |    **说明**    |
|    ----    |    ----    |    ----    |
|    1    |    菜单栏    |    菜单栏主要包括工作台、设计中心、可视化学院、帮助和教程、跳转至可视化应用中心、账户管理、用户中心、主题切换等。    |
|    2    |    工作空间    |    当前区域为可视化工作台中工作空间的管理部分，包括空间分组管理和空间配额管理等。详情请参见[工作空间概述](/zh/datav/datav-7-0/user-guide/workspace-overview)。    |
|    3    |    项目分组    |    当前区域为可视化工作台中我的项目分组展示部分，包括对项目分组的增删改查以及回收站等功能。详情请参见[项目分组管理](/zh/datav/datav-7-0/user-guide/manage-project-groups)。    |
|    4    |    功能导航    |    当前区域为可视化工作台中特色功能的集中展示部分，包括数据集/数据源/过滤器的管理、组件包的开发、区块的设计及管理、组件收藏管理以及设计资源的集中管理等功能。数据准备功能，请参见[数据源分类](/zh/datav/datav-7-0/user-guide/data-source-categories)。组件开发功能，请参见[组件开发快速入门](/zh/datav/datav-7-0/user-guide/get-started-with-widget-development-of-datav-7)。资源管理功能，请参见[设计资产概述](/zh/datav/datav-7-0/user-guide/overview-of-design-resources)。    |
|    5    |    创建看板    |    当前区域为可视化工作台中的看板创建部分，包括​创建PC端看板、创建移动端看板和创建三维城市功能。创建PC端看板，请参见[使用模板创建PC端看板](/zh/datav/datav-7-0/user-guide/use-a-template-to-create-a-project-on-pc)。​创建移动端看板，请参见[使用模板创建移动端看板](/zh/datav/datav-7-0/user-guide/use-a-template-to-create-a-project-on-a-mobile-device)。创建三维城市，请参见[创建三维城市](/zh/datav/datav-7-0/user-guide/create-a-3d-city)。    |
|    6    |    数据看板    |    当前区域为可视化工作台中的数据看板管理部分，包括预览、发布、编辑、复制、拷屏、拷屏到工作空间、移动到项目分组、重命名和删除等操作。详情请参见[管理数据看板](/zh/datav/datav-7-0/user-guide/manage-a-data-dashboard)。​    |
|    7    |    三维城市    |    当前区域为可视化工作台中的三维城市管理部分，包括预览、编辑、复制、移动到项目分组、重命名和删除等操作。详情请参见[管理三维城市](/zh/datav/datav-7-0/user-guide/manage-a-3d-city)。    |
|    8    |    宣传视频    |    当前区域为可视化工作台中宣传视频或内容展示区域，单击即可查看详情。    |
|    9    |    快速开始    |    当前区域为可视化工作台中新手操作教程部分，可以通过查看教学视频，快速学习产品的主要功能，快速上手可视化看板创作。    |
|    10    |    答疑中心    |    当前区域为可视化工作台中的答疑区，单击即可重播新手引导、跳转官网文档中心、提交工单和联系钉钉答疑群。    |

### **设计中心**

设计中心展示了系统为您提供的各式各样的设计模板和套件，您可按照需求选择设计模板或套件快速构建自己的数据看板。![2.jpg](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4571419861/p692809.jpg)![3.jpg](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4571419861/p692810.jpg)

|    **序号**    |    **区域名称**    |    **说明**    |
|    ----    |    ----    |    ----    |
|    1    |    选择模板    |    当前区域是设计中心的模板选择区域，可以通过应用场景、设计风格和颜色维度，筛选需要的大屏模板。    |
|    2    |    使用模板    |    当前区域为设计中心的模板使用区域，可以预览看板和以当前模板创建新的数据看板。详情请参见[使用模板创建PC端看板](/zh/datav/datav-7-0/user-guide/use-a-template-to-create-a-project-on-pc)和[使用模板创建移动端看板](/zh/datav/datav-7-0/user-guide/use-a-template-to-create-a-project-on-a-mobile-device)。    |
|    3    |    套件管理    |    当前区域为设计中心中的可视化主题套件使用区域，共有晴山蓝、丹霞橙、暮山紫、碧玺绿和云峰白等套件主题选择，每种主题包含套件模板、数据看板、设计规范、地图、标题&素材等部分。详细使用请参见[设计库](/zh/datav/datav-7-0/user-guide/design-library)。    |

## **可视化应用中心**

[可视化应用中心](https://datav.aliyun.com/v/appcenter/243895)让您可以定制数据看板演示序列，支持在网页端和客户端投屏展示您的数据看板。

![1.jpg](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0854419861/p692892.jpg)

|    **序号**    |    **区域名称**    |    **说明**    |
|    ----    |    ----    |    ----    |
|    1    |    工作空间    |    当前区域为可视化应用中心中的工作空间管理部分，通过切换工作空间，快速进入不同的可视化应用中心，查看不同的预案。单击管理工作空间即可进入工作空间管理页面，详情请参见[工作空间概述](/zh/datav/datav-7-0/user-guide/workspace-overview)。    |
|    2    |    菜单栏    |    当前区域为可视化应用中心中的菜单栏部分，包括跳转至可视化工作台和查看账户等功能。    |
|    3    |    预案展示    |    当前区域为可视化应用中心中演示预案的展示页面，包括预案的投屏演示和查看全部功能。    |
|    4    |    我的看板    |    当前区域为可视化应用中心中数据看板的展示页面，当可视化工作台中的数据看板发布后，此处即展示已发布的数据看板。    |

## **可视化演播厅**

在可视化应用中心点击右侧最近使用的查看全部，即可进入可视化演播厅。您可以创建演示预案，快速管理您的所有演示预案。![1.jpg](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0854419861/p692860.jpg)![2.jpg](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0854419861/p692864.jpg)

|    **序号**    |    **区域名称**    |    **说明**    |
|    ----    |    ----    |    ----    |
|    1    |    工作空间    |    当前区域为可视化演播厅中工作空间的管理部分，通过切换工作空间，快速进入不同的可视化应用中心，查看不同的预案。单击管理工作空间即可进入工作空间管理页面，详情请参见[工作空间概述](/zh/datav/datav-7-0/user-guide/workspace-overview)。    |
|    2    |    菜单栏    |    当前区域为可视化演播厅中的菜单栏部分，包括打开投屏播放器和查看账户等功能。    |
|    3    |    管理预案    |    当前区域为可视化演播厅中演示预案的管理页面，包括创建预案、投屏预案、预览预案、编辑预案、删除预案和管理配额等操作。详情请参见[管理预案](/zh/datav/manage-presentations)。    |
|    4    |    快速开始    |    当前区域为可视化演播厅中新手操作教程部分，通过查看教学步骤，快速使用演示预案功能。详情请参见[投屏预案](/zh/datav/mirror-the-screen-of-presentations-on-the-web-page)。    |
|    5    |    快速预览    |    当前区域为可视化演播厅中演示预案的预览部分，可以查看当前预案的基本信息，投屏、编辑和删除预案。    |
