OD飞线是三维城市构建器的图层，支持独立的样式和数据配置，能够以飞线的形式表现地理位置上两个点之间的OD数据，可以用来展示两地之间的关联信息，如物流、交易等。本文介绍OD飞线各配置项的含义。

在场景编辑器左侧图层列表中，单击OD飞线图层，进入配置面板配置OD飞线图层的样式和数据。 

**说明** 如果图层列表中没有OD飞线，说明您还未添加OD飞线图层，请参见[添加子组件](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/layer-panel-management-of-professional-edition#section-1yb-xd4-pfe)进行添加。 

## 配置

![od飞线层配置面板](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0217548461/p421913.png)
* 搜索配置：单击配置面板右上角的搜索配置，可在搜索配置面板中输入您需要搜索的配置项名称，快速定位到该配置项，系统支持模糊匹配。详情请参见[搜索资产配置](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/search-for-configurations#task-2376842 "通过搜索资产配置功能，您可以快速定位到资产的对应配置项，在搜索配置页面直接修改配置样式，也可以复用同类型的配置样式。")。
* OD飞线层配置面板参数说明：  
  * 通用：通用配置参数说明。  

|    参数    |    说明    |
|    ----    |    ----    |
|    混合模式    |    设置OD飞线层的混合叠加渲染模式，可选添加模式、正常模式和无叠加模式。    |
|    随距衰减    |    打开开关，飞线的屏幕空间宽度随着与相机距离的增加而逐渐减小；关闭开关，飞线的屏幕空间宽度保持不变。    |
|    飞线分段    |    一条飞线由许多条直线段构成，您可以拖动滑块，调整飞线分段的数值，该值越大，飞线越平滑，性能开销也越大，取值范围为1\~300。    |
|    边缘模糊    |    调节飞线边缘的锯齿优化效果，该值越大，抗锯齿效果越好，但视觉上飞线也会越细。您可以拖动滑块，设置飞线边缘的模糊比例，取值范围0\~1。    |
  * 图形：图形配置面板参数说明。  

|    参数    |    说明    |
|    ----    |    ----    |
|    飞线弧度    |    拖动滑块设置飞线弧度矢跨比的值，取值范围为0\~10。    |
|    飞线线型    |    设置飞线线型样式，可选矢量和贴图。    |
|    基底线    |    设置基底线样式。 基线粗细：拖动滑块，调整基底线的粗细值，取值范围0\~500。 基线颜色：设置基底线的颜色。 不透明系数：拖动滑块，调整基底线的不透明系数，取值范围0\~1。    |
|    高亮线    |    设置高亮线样式。仅在飞线线型选择矢量时显示。 高亮数量：拖动滑块，调整高亮线的数量值，取值范围1\~30。 高亮粗细：拖动滑块，调整高亮线的粗细值，取值范围0\~200。 虚线长度：拖动滑块，调整虚线的长度值，取值范围0\~1。 高亮颜色：设置高亮线的颜色。 不透明系数：拖动滑块，调整高亮线的不透明系数，取值范围0\~1。    |
|    飞线贴图    |    设置飞线贴图的样式。仅在飞线线型选择贴图时显示。 飞线：贴图类型为默认时，设置飞线的默认贴图类型和飞线在u方向及v方向的平铺值；贴图类型为自定义时，设置自定义飞线贴图的地址和飞线在u方向及v方向的平铺值。 符号贴图：贴图类型为默认时，设置飞线的默认符号贴图类型和贴图之间的符号间距；贴图类型为自定义时，设置自定义飞线贴图的地址和贴图之间的符号间距。    |
  * 动效：动效配置面板参数说明。  

|    参数    |    说明    |
|    ----    |    ----    |
|    飞线速度    |    拖动滑块，调整飞线的移动速度，取值范围为-0.2\~0.2，其中数值的正负表示飞线的运动方向。    |
|    符号速度    |    拖动滑块，调整飞线的符号速度，取值范围为-0.04\~0.04，其中数值的正负表示符号运动的方向。    |

## 蓝图编辑器交互配置

1. 在当前场景编辑器页面，单击左侧图层列表中OD飞线右侧的![导出到蓝图编辑器](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2434449951/p89089.jpg)图标。
2. 单击页面左上角的![蓝图图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2434449951/p89087.jpg)图标。
3. 在蓝图编辑器配置页面，单击导入节点栏内的OD飞线图层，在画布中可以看到如下图所示的OD飞线的蓝图编辑器配置参数。![od飞线蓝图配置参数](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0217548461/p421943.png)
  * 事件

|    事件    |    说明    |
|    ----    |    ----    |
|    当数据接口请求完成时    |    数据接口请求返回并经过过滤器处理后抛出的事件，同时抛出处理后的JSON格式的数据。具体数据示例请参见画布编辑器中组件右侧配置面板数据页签的数据响应结果区域。    |
  * 动作

|    动作    |    说明    |
|    ----    |    ----    |
|    请求数据接口    |    重新请求服务端数据，上游数据处理节点或图层节点抛出的数据将作为参数。例如OD飞线配置了API数据源为http://api.test，传到请求数据接口动作的数据为{ id: '1'}，则最终请求接口为http://api.test?id=1。    |
|    导入数据接口    |    按组件绘制格式处理数据后，导入组件，重新绘制。不需要重新请求服务端数据。具体数据示例请参见画布编辑器中组件右侧配置面板数据页签的数据响应结果区域。    |
|    显示组件    |    显示组件，不需要参数。    |
|    隐藏组件    |    隐藏组件，不需要参数。    |
