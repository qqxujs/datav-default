本文介绍画布编辑器中的历史记录功能。

**说明** 

历史记录功能为7.0版本产品新增功能，请根据[新版控制台](https://datav.aliyun.com/v/console/)环境进行操作。

**重要** 
* 为避免多用户操作时互相干扰，目前历史记录会在页面刷新后清空，如果需要回退历史操作，请确保页面不被刷新。
* 目前阿里云控制台在3小时用户无操作的情况下会自动登出，登出重进会刷新页面，导致历史记录清空，您可以进入[阿里云控制台](https://www.aliyun.com/)，单击右上角的账号，进入账号中心，选择账号安全，修改登录保持时间，以避免非预期的自动登出。

## **前提条件**

* 已登录DataV控制台
* 已进入画布编辑器页面

## **回退到对应记录**

1. 在左侧导航栏单击**历史记录**，进入历史记录页签。
2. 鼠标hover到需要回退到的记录上，单击鼠标左键。
3. 系统会直接回退到当前记录时的画布编辑器状态。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5179649761/p607743.png)

## **清空历史记录**

1. 在左侧导航栏单击**历史记录**，进入历史记录页签。
2. 单击历史记录列表上方的**清空**。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5179649761/p607745.png)
