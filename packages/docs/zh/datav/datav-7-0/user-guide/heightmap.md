高度图地形瓦片是三维城市构建器的全局图层，支持独立的样式和数据配置，包括高度图地形瓦片的通用配置和交互配置样式。本文介绍高度图地形瓦片配置项的含义。

在场景编辑器左侧图层列表中，单击**高度图地形瓦片**图层，进入配置面板配置**高度图地形瓦片**图层的样式和数据。 

**说明** 
* 当前组件目前仅供**DataV尊享版**用户使用。如需使用该组件，请先将版本升级至尊享版。
* 如果图层列表中没有**高度图地形瓦片**，说明您还未添加**高度图地形瓦片**图层，请参见[添加子组件](https://help.aliyun.com/zh/datav/datav-7-0/user-guide/layer-panel-management#section-1yb-xd4-pfe)进行添加。

## 样式面板

![配置面板](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6326448461/p423340.png)
* **搜索配置**：单击**配置**面板右上角的**搜索配置**，可在搜索配置面板中输入您需要搜索的配置项名称，快速定位到该配置项，系统支持模糊匹配。详情请参见[搜索配置项](https://help.aliyun.com/zh/datav/datav-7-0/user-guide/configuration-items#b14c340069fac)。
* **通用**

| **参数**      | **说明**                                                                                                                                                                           |
| ----------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **投影类型**    | 设置高度图地形瓦片的投影类型。                                                                                                                                                                  |
| **BaseURL** | 设置高度图地形瓦片的BaseURL，需包含 {x} 、{y}、{z} 占位符。                                                                                                                                          |
| **瓦片配置**    | 设置高度图地形瓦片的配置项。**宽度**：设置高度图地形瓦片的宽度。**高度**：设置高度图地形瓦片的高度。**最小缩放**：拖动滑块设置高度图地形瓦片的最小缩放倍数，取值范围为0\~20。**最大缩放**：拖动滑块设置高度图地形瓦片的最大缩放倍数，取值范围为0\~20。**缩放偏移**：拖动滑块设置高度图地形瓦片的缩放偏移值，取值范围为-3\~3。 |
* **交互**

| **参数**   | **说明**                   |
| -------- | ------------------------ |
| **鼠标交互** | 打开开关，开启鼠标交互；关闭开关，关闭鼠标交互。 |

## 数据面板

此图层不需要配置数据。

## 蓝图交互

1. 在当前场景编辑器页面，单击左侧图层列表中**高度图地形瓦片**右侧的![导出到蓝图编辑器](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2434449951/p89089.jpg)图标。
2. 单击页面左上角的![蓝图图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2434449951/p89087.jpg)图标。
3. 在蓝图编辑器配置页面，单击导入节点栏内的**高度图地形瓦片**图层，在画布中可以看到如下图所示的高度图地形瓦片的蓝图编辑器配置参数。![蓝图配置参数](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6326448461/p423349.png)

  * **事件**

| **事件**   | **说明**                              |
| -------- | ----------------------------------- |
| **左键点击** | 当鼠标左键单击高度图地形瓦片时抛出的事件，抛出数据为object类型。 |
  * **动作**

| **动作**   | **说明**      |
| -------- | ----------- |
| **显示组件** | 显示组件，不需要参数。 |
| **隐藏组件** | 隐藏组件，不需要参数。 |
