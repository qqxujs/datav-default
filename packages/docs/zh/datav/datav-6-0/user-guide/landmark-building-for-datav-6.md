本文介绍空间档案页面中地方标志性建筑和精模实体的基本属性信息。

单击空间档案页面中的地标建筑，即可打开地标建筑和精模实体的详情页面。![地标建筑入口](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0910342561/p439236.png)

## 地标建筑管理

在地标建筑页面中，您可以进行如下功能操作： 

|    功能    |    操作说明    |
|    ----    |    ----    |
|    查看地标建筑详情信息    |    在地标建筑页面中，会显示当前系统内所有地标建筑的内容。您可以单击某个建筑处任何位置或最右侧的![打开列表](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8370270261/p243480.jpg)图标，即可打开建筑信息详情列表查看地标建筑的基本信息和模型信息，详情请参见[地标建筑信息](#section-p99-bht-51d)和[模型信息](#section-3kl-zms-zrr)。![打开列表查看详情](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8370270261/p243503.png)    |
|    搜索地标建筑    |    您可以单击页面上方的搜索地标输入框，自定义输入地标建筑名称后，单击右侧的![搜索框](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5110270261/p243433.jpg)即可查找到所需的地标建筑内容。![搜索地标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8370270261/p243513.png)    |

## 地标建筑信息

单击选择某个地标建筑后，您可以查看到该地标建筑的基本信息，包括建筑类型、建筑年代和详细地址。 

您可以单击详细地址右侧的![地图视角](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9370270261/p243567.jpg)图标，在查看地图对话框中，查看当前地标建筑的具体地图样式。![查看地图](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9370270261/p243568.jpg)

## 模型信息

打开某个地标建筑详情列表后，您可以看到该建筑的模型信息详情，包括模型风格、模型等级、模型大小和球面坐标系。 

您可以单击模型信息框最右侧的![预览图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9370270261/p243580.jpg)图标预览模型，在弹出的预览模型对话框中，您可以进行以下操作。![模型参数面板](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9370270261/p243599.jpg)
* 查看模型基本信息：在对话框左上角处，您可以查看当前被预览的模型的模型等级、风格、制作商、文件大小和上传时间。![模型基本信息](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9370270261/p243797.jpg)
* 配置模型参数：在对话框最右侧，您可以查看并配置模型参数样式。  
配置模型参数说明：  
  * 贴图通道：单击模型的贴图通道内最终渲染、基本颜色、金属色、粗糙度、法线、发光和AO，即可查看模型不同配置项的样式
  * 环境光强度：在模型预览页面， 您可以拖动滑块调整模型的环境光强度效果，数值越大，亮度越大。
  * 显示：模型的显示样式。  

|    参数    |    说明    |
|    ----    |    ----    |
|    背景    |    选中背景配置项后，自定义选择设置模型背景的颜色或环境贴图。 颜色：设置背景的颜色。![](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9370270261/p243790.png) 环境贴图：您可以输入图片的URL地址或单击图片区域内的更改上传图片，也可以单击图片区域删除当前图片后再次单击上传本地图片作为模型的背景贴图。    |
|    强度    |    您可以拖动滑块调整模型的2环贴图境强度，数值越大，亮度越大。    |
|    开关    |    打开开关，显示设置的环境作为模型背景；关闭开关，不显示环境贴图样式。    |
* 保存封面：在页面的中间模型预览区域右上角，您可以在配置完参数后，单击![保存封面](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9370270261/p243801.jpg)图标，即可保存当前参数配置下的模型作为预览封面。
