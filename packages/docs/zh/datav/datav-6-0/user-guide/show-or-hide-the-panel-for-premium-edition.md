  管理场景编辑器中的面板显隐-DataV数据可视化-阿里云

场景编辑器内工具栏提供了切换编辑器内各模块面板显隐的功能。通过该功能，您可以随意切换隐藏打开各面板，更直观方便地查看场景编辑器内的布局内容。

## 使用方法

1.  在场景编辑器页面内，鼠标移动到页面左上角的五个切换面板显隐按键。![切换显隐](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1516197461/p416041.png)
    
    **说明** 默认情况为五个面板显示全部为打开状态。
    
2.  单击![子组件](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8528786061/p186613.png)图标，即可关闭场景编辑器左侧的图层管理面板。
3.  单击![关键帧](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8528786061/p186615.png)图标，即可关闭场景编辑器下方的分镜与关键帧管理面板。
4.  单击![右侧面板](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8528786061/p186616.png)图标，即可关闭场景编辑器右侧的场景配置面板或图层配置面板。
5.  单击![相机路劲面板](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1516197461/p416047.jpg)图标，即可关闭场景编辑器中间的相机路径面板。
6.  单击![相机高程面板](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1516197461/p416052.jpg)图标，即可关闭场景编辑器中间的相机高程面板。
7.  再次单击五个置灰的图标，被关闭的面板重新显示在场景编辑器界面内。![置灰面板](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1516197461/p416057.jpg)
