本文为您介绍开通DataV-Board 7.0（数据看板）服务的流程。

## **购买须知**

* DataV-Board产品包括**个人版、企业版**、**专业版**和**尊享版**等正式版本，以及供您体验的试用版本（即**专业版试用**），各版本支持购买的时长及功能存在差异。购买前请先参考[规格说明](https://help.aliyun.com/zh/datav/datav-7-0/product-overview/datav-editions)、[产品计费](https://help.aliyun.com/zh/datav/datav-7-0/product-overview/product-billing-datav-7/)，了解产品的版本功能及计费标准。
* **专业版试用**仅支持新用户购买，且每位新用户只能购买一次（购买后可使用一个月）。

## **开通服务**

DataV-Board的开通流程如下：

1. 进入[DataV-Board服务开通页](https://common-buy.aliyun.com/?commodityCode=datav)。
2. 选择需要购买的产品**版本**和**购买时长**，单击**立即购买**。
3. 在**确认订单**页面，确认购买信息并勾选服务协议，单击**去支付**。
4. 在**支付**页面，查看订单，确认无误后，按照界面指引订购。

## 后续步骤

购买成功后，可进入[DataV控制台](https://datav.aliyun.com/)，使用[模板搭建数据看板](https://help.aliyun.com/zh/datav/datav-7-0/getting-started/use-a-template-to-create-a-dashboard)开始体验产品。
