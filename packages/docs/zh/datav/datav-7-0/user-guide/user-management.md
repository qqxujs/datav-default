本文为您介绍如何查看用户数使用情况、账号移除等相关操作。

## **用户管理基本操作**

用户管理的主要操作如下。

| 操作      | 说明                         |
| ---- | ---- |
| 用户数使用概览 | 当前实例下用户数配额数量及使用情况。         |
| 账号信息列表  | 展示实例的所有的账号信息及关联的工作空间、账号状态。 |
| 账号移除    | 移除所有工作空间当前账号。              |

## **使用限制**

不同版本DataV-Board 7.0默认支持的用户数如下。

| **DataV-Board 7.0版本** | **用户数量** |
| ---- | ---- |
| **专业版试用**             | 1个       |
| **个人版**               |          |
| **企业版（新）**            | 2个       |
| **专业版（新）**            | 5个       |
| **尊享版（新）**            | 100个     |

**说明** 

企业版及以上版本已经支持用户数扩展，当用户数到达上限时，您可以进入DataV控制台，单击右上角账号下拉的去升级， 进入升配页购买页面内的**用户数资源包**。

## **前提条件**

* 已[开通DataV-Board服务](https://help.aliyun.com/zh/datav/datav-7-0/getting-started/activate-datav-1)。
* （可选）[创建RAM用户](https://help.aliyun.com/zh/ram/user-guide/create-a-ram-user)。若工作空间需要添加成员进行协同开发，请先创建相应RAM用户。

## **进入工作空间管理**

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 单击左上角工作空间名称后的![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9953391471/p927476.png)图标，选择**管理**，进入用户管理页面。

## **用户管理**

进入用户管理页面，展示用用户数情况、账号明细列表，可执行如下操作：

* **查看用户数配额使用情况：**展示当前实例下用户数配额及已使用情况。  
![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0473976571/p973803.png)
* **搜索用户：**可按照子账户用户名、子账号ID，搜索目标账号。  
![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0473976571/p973814.png)
* **查看账号详细信息：**可查看实例下的账号、账号ID、关联的工作空间、账号状态。  
![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0473976571/p973810.png)
* **移除**：单击移除按钮，可将当前账号移除所关联的工作空间。

**说明** 

账号状态，包括正常、过期未释放、释放。
