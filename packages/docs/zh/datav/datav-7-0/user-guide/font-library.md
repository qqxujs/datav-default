本文为您介绍DataV-Board 7.0（数据看板）支持的字体及字体的相关配置操作。

## **字体介绍**

字体即DataV-Board 7.0中组件文本的显示样式。DataV-Board 7.0支持使用多种官方字体（例如，阿里巴巴普惠体、钉钉进步体）及本地安装的字体（例如，楷体、微软雅黑），本地字体需安装后才会生效。未安装的字体会呈现为失效状态（即显示![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8890769271/p862441.png)图标），若选择未安装的字体，则组件文本会自动转换为系统默认的字体样式。

## **字体相关配置**

以“通用标题”组件为例，为您介绍字体的相关配置。不同组件涉及的配置操作存在差异，具体请以实际界面为准。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8808379271/p862518.png)

核心参数如下。

| **配置项** | **描述**                                                                                                                           |
| ---- | ---- |
| 字体      | 文本的显示样式。单击字体下拉框即可选择所需样式，也可按需搜索目标字体。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8808379271/p862538.png) |
| 粗细      | 文字的粗细，包括标准、粗体、更粗、更细，也可配置为100\~900范围内的数值。                                                                                         |
| 字号      | 文字的大小。最大配置为1000px。                                                                                                               |
| 颜色      | 文字的颜色，按需调整。                                                                                                                      |
| 阴影      | 在文字上添加仿真阴影效果的设计元素，赋予文本立体感，增加视觉层次。可设置阴影颜色、模糊半径、偏移量。                                                                               |
| 布局      | 文字在组件内的位置分布、排列方向及间隔大小。                                                                                                           |

## 字体的配置**入口**

您可单独配置某个组件的字体，也可批量统一看板中多个组件的字体。

### **配置单个组件字体**

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 进入目标看板的画布编辑页面。
3. 单击目标组件，在右侧**样式**页签配置字体信息。

**说明**  
在**样式**页签中，不同组件设置字体的位置可能存在差异，具体请以实际界面为准，或参考具体[组件](https://help.aliyun.com/zh/datav/datav-7-0/user-guide/widgets-on-pcs/)的配置页面。
