颜色映射器适用于设置组件内各种颜色和数据数值之间的映射配置样式，内置十余种颜色模板，同时支持自定义颜色和颜色的对应值，以满足用户不同的定制需求。本文以配置**基础平面地图 3.0**组件中**矢量散点层**子组件的散点颜色为例，为您介绍如何使用颜色映射器实现数据与颜色之间的映射匹配。

## 配置说明

配置地图子组件内散点颜色时，您可以通过以下两种颜色配置方法调整散点的颜色。

* 方法一：单击关闭**散点颜色**配置项左侧的![关闭颜色映射](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4553257261/p260339.jpg)图标，在鼠标悬浮态提示**开启视觉映射**时，使用[颜色选择器](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/configure-item-description#section-kdw-vj4-t2b)，无需数据驱动，单纯的自定义样式配置，进行单个颜色调整。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3365781861/p633886.png)
* 方法二：单击打开**散点颜色**配置项左侧的![打开状态](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4553257261/p260340.jpg)图标，在鼠标悬浮态提示**关闭视觉映射**时，使用颜色映射器，由散点的数据驱动映射，可对散点颜色进行多选配置。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3365781861/p633888.png)

## 映射类型

颜色映射类型分为三大类：**名词分类**型、**数值连续**型和**数值分区**型。

| **类型**   | **说明**                                                                                                                                                                                                                                                                                                                   |
| ---- | ---- |
| **名词分类** | 适用于需要给名称和类别等分类型的数据，提供颜色上的区分。色板上各个颜色之间互相区分度较大。图 1\. 名词分类颜色模板样式![名词分类颜色模板样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6763257261/p267952.jpg)图 2\. 名词分类型数据示例图![名词分类型示例图](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6763257261/p260592.png)                                          |
| **数值连续** | 适用于给数值型数据分配连续渐变颜色的情况，且数值和颜色在映射空间内为一一对应的关系。图 3\. 数值连续颜色模板样式![数值连续颜色模板样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6763257261/p267953.jpg)图 4\. 数值连续型数据示例图![数值连续型示例图](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6763257261/p260596.png)                                             |
| **数值分区** | 适用于在数值连续型映射的基础上，对连续的数值进行分区，将同一数值区间内的数据归为一类，并将此区间内的数据填充相同颜色的情况。使用数值分区型映射，可以减少颜色数量，避免视觉混乱。图 5\. 数值分区颜色模板样式![数值分区颜色模板样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6763257261/p267954.jpg)图 6\. 数值分区数据示例![数值分区数据示例](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7763257261/p267955.png) |
