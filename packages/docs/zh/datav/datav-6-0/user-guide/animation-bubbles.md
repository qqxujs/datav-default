  三维城市构建器呼吸气泡图层配置指南-DataV数据可视化-阿里云

呼吸气泡是三维城市构建器的图层，可以表示点状要素的位置和属性。支持独立的样式和数据配置，包括呼吸气泡的颜色、缩放倍数、转动速度、亮度和高度偏移等。本文介绍呼吸气泡配置项的含义。

在场景编辑器左侧图层列表中，单击呼吸气泡图层，进入配置面板配置呼吸气泡图层的样式和数据。

**说明** 如果图层列表中没有呼吸气泡，说明您还未添加呼吸气泡图层，请参见[添加子组件](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/layer-panel-management-of-professional-edition#section-1yb-xd4-pfe)进行添加。

## 配置

-   搜索配置：单击配置面板右上角的搜索配置，可在搜索配置面板中输入您需要搜索的配置项名称，快速定位到该配置项，系统支持模糊匹配。详情请参见[搜索资产配置](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/search-for-configurations#task-2376842 "通过搜索资产配置功能，您可以快速定位到资产的对应配置项，在搜索配置页面直接修改配置样式，也可以复用同类型的配置样式。")。
-   呼吸气泡配置面板参数说明。
    -   通用
        
         
        
        参数
        
        说明
        
        混合模式
        
        设置呼吸气泡层的混合叠加渲染模式，可选添加模式和正常模式。
        
        深度写入
        
        打开开关，呼吸气泡层可进行深度写入；关闭开关，呼吸气泡层不可进行深度写入。
        
        **说明** 深度写入打开后会可能导致该对象遮挡其后面的对象。
        
        深度测试
        
        打开开关，呼吸气泡层可进行深度测试；关闭开关，呼吸气泡层不可进行深度测试。
        
        **说明** 深度测试打开可能后会导致该对象被其后面的对象遮挡。
        
        缩放倍数
        
        呼吸气泡时的缩放倍数值，倍数越大尺寸越大，取值范围为0~1000。
        
        高度偏移
        
        呼吸气泡区域距离地平面高度的偏移值 ，取值范围为-3000~5000。

## 蓝图编辑器交互配置

1.  在当前场景编辑器页面，单击左侧图层列表中呼吸气泡右侧的![导出到蓝图编辑器](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2434449951/p89089.jpg)图标。
2.  单击页面左上角的![蓝图图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2434449951/p89087.jpg)图标。
3.  在蓝图编辑器配置页面，单击导入节点栏内的呼吸气泡图层，在画布中可以看到如下图所示的呼吸气泡的蓝图编辑器配置参数。![呼吸气泡蓝图配置参数](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2909548461/p422409.png)
    -   事件
        
         
        
        事件
        
        说明
        
        当数据接口请求完成时
        
        数据接口请求返回并经过过滤器处理后抛出的事件，同时抛出处理后的JSON格式的数据。具体数据示例请参见画布编辑器中组件右侧配置面板数据页签的数据响应结果区域。
        
        渲染完成
        
        呼吸气泡渲染完成时抛出的事件，同时抛出object类型的数据。
        
    -   动作
        
         
        
        动作
        
        说明
        
        请求数据接口
        
        重新请求服务端数据，上游数据处理节点或图层节点抛出的数据将作为参数。例如呼吸气泡配置了API数据源为`http://api.test`，传到请求数据接口动作的数据为`{ id: '1'}`，则最终请求接口为`http://api.test?id=1`。
        
        导入数据接口
        
        按组件绘制格式处理数据后，导入组件，重新绘制。不需要重新请求服务端数据。具体数据示例请参见画布编辑器中组件右侧配置面板数据页签的数据响应结果区域。
        
        显示组件
        
        显示组件，不需要参数。
        
        隐藏组件
        
        隐藏组件，不需要参数。
