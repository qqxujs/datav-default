本文为您介绍使用欢乐砸金蛋模板创建可视化应用的方法。

## 前提条件

已开通DataV服务，详情请参见[开通DataV服务](https://help.aliyun.com/zh/datav/datav-6-0/getting-started/activate-datav#task-2323429 "本文为您介绍开通DataV服务的方法。")。 

## 操作步骤

1. 登录[DataV控制台](https://datav.aliyun.com/)。
2. 在我的可视化页面中，单击PC端创建。  
![新建可视化](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2124968461/p10314.png)
3. 您可以在模板页面内浏览并选择欢乐砸金蛋可视化应用模板，单击创建项目。  
![欢乐砸金蛋大屏入口](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1359962161/p238945.png)
4. 在创建数据大屏对话框中，输入可视化应用名称，并设置应用的分组。
5. 单击创建。  
应用创建成功后会跳转到画布编辑器页面，即可看到一款满足年会或团建砸金蛋功能且设计精良的模板。![欢乐砸金蛋模板大屏画布编辑器](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1359962161/p238948.jpg)
