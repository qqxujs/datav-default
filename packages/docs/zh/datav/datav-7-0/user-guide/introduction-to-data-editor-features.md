 

数据编辑器是一款支持多种地理数据编辑与管理功能的小工具，本文介绍数据编辑器的详细功能。

## 进入数据编辑器

单击[DataV.GeoAtlas地理小工具系列](https://datav.aliyun.com/portal/school/atlas/area_generator)，打开**数据编辑器**工具页面。

## 界面概述

数据编辑器页面分为三大主要功能面板，分别为**地图面板**、**数据表面板**和**GeoAtlas智能助理**。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p965764.png)

名称

说明

[地图面板](#section-vpe-mq0-hxi)

地图面板是数据编辑器的核心交互界面，集成地图浏览与几何编辑功能，通过顶部/左侧双工具栏设计提供高效操作体验。

2

[数据表面板](#565870d25217f)

数据表面板作为数据编辑器的核心管理界面，提供从数据上传、编辑到导出的全流程处理能力，支持多种地理数据格式的导入与标准化输出，通过结构化设计实现地理数据的高效管理与操作。

3

[GeoAtlas智能助理](#91a556d4ae4ik)

GeoAtlas智能助理是数据编辑器内置的智能交互工具，通过自然语言交互帮助用户快速完成地图任务，包括路线规划、地点查询、数据模拟与分析等功能，显著提升使用效率。

## 地图面板

地图面板采用顶部编辑栏与左侧管理栏的双工具栏设计：顶部工具栏专注于地图要素编辑功能，左侧工具栏提供数据查看与导出等操作，配合全局快捷键支持，实现可视化编辑与数据管理的高效协同作业。

-   **单选**：单选模式时单击要素即可选中，依次单击可以选中多个，再次单击选中的数据可取消选中。
    
-   **框选**：地图框选操作，批量选择图形区域内的要素，包含**矩形框选**、**圆形框选**、**多边形框选**。

绘制要素

支持绘制**点**、**线**、**矩形**、**圆形**、**多边形**要素。单击对应图标，进入数据绘制状态，键盘`Esc`键可以取消绘制状态。绘制完成后，页面下方数据表面板会显示对应要素数据。![2025-06-10_13-31-54](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p965874.gif)

要素编辑工具

要素编辑工具包含**要素平移**、**要素旋转和缩放**、**要素顶点编辑**功能。单击对应图标，进入对应编辑状态，地图没有数据时编辑工具为置灰状态。![2025-06-10_13-53-44](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p965920.gif)

要素空间关系

要素空间关系包含**几何合并**、**几何交集**、**几何差集**功能。选中要素数量少于两个时几何工具为置灰状态。

**说明**

同类型要素特指相同几何类别的要素匹配，包括点与点、线与线、面与面的组合，其中面类型要素涵盖矩形、圆形及多边形等闭合图形。

-   **几何合并**：选中两个及以上**同类型要素**后单击![{1E1C8E33-002F-4668-A79D-E63C1B5F4DBA}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p965942.png)图标合并要素，合并后的要素会成为一个整体，若这些要素有相交处则融合成为一个新的多边形要素。
    
-   **几何交集**：选中两个及以上**面类型要素**后单击![{0A63C948-1939-4C8D-8A2C-2DB94AE4D8EE}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p965946.png)图标执行面数据交集计算操作。开启**跳过点和线**会忽略选中的点和线数据。
    
-   **几何差集**：选中两个及以上**面类型要素**后单击![{E7939A87-DBF5-4D7B-AA04-0681E84FD941}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p965948.png)图标执行面数据差集计算操作。开启**跳过点和线**会忽略选中的点和线数据。
    

要素处理工具集

要素处理工具集包括**要素打散**、**要素剪裁**、**要素简化**、**面内生成随机点**、**沿线生成随机点**功能，未选中数据时处于置灰状态。

-   **要素打散**：要素打散功能用于将复合几何类型（MultiPolygon/MultiLineString/MultiPoint）拆解为对应的独立要素（Polygon/LineString/Point），这是要素几何合并的逆向操作，适用于需要对复合要素中的单个几何体进行独立处理的场景。示例如下。
    
    要素打散前：![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p965979.png)
    
    要素打散后：![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p965983.png)
    
-   **要素剪裁**：选中一个或多个要素后单击![{46FEEB32-FC89-45FE-B67E-87BBCAA4515E}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p965989.png)图标，在弹窗中选择**点击绘制裁剪路径**，按照一定的路径进行分割裁剪操作，绘制完成后，输入框中会显示路径数据，单击**执行**完成裁剪。原始要素将按路径分割为多个独立可编辑的新要素。
    
    **说明**
    
    当裁剪路径完全位于单个要素内部时，该路径将被判定为无效裁剪路径。
    
-   **要素简化**：用于将面数据通过一定的规则减少顶点数量、细节，保留关键特征。选中要素后单击![{C747FBA9-6C1C-4F3D-98AF-BAE08A6F4CC2}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p966002.png)图标打开弹窗，配置**简化容差**和**简化质量**后单击**执行**。
    
    -   简化容差：控制简化的粗糙度，容差越大，简化程度越高，几何形状越粗糙。
        
    -   简化质量：普通简化以速度优先，牺牲部分几何精度；高质量简化在保持形状特征的前提下，追求更高精度和视觉效果。
        
-   **面内生成随机点**：面内随机生成点用于在给定的一个或者多个面内随机或均匀分布点数据。选中面要素后单击![{F236F6B4-4BF6-4FD1-A31E-4965ECC9A4B8}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p966009.png)图标打开弹窗，配置信息后单击**执行**。
    
    -   最大数量：单个面内最多生成的数据量上限。
        
    -   分布规则：支持随机分布、聚类分布、均匀分布三种规则。
        
        -   随机分布：区域内完全随机分布点。
            
        -   聚类分布：点要素按照某种聚集模式而非完全随机地分布，并在数据表中生成一个`value`字段用于模拟空间集聚现象。
            
        -   均匀分布：在区域内，点要素尽可能保持彼此间距均衡、避免聚集地随机生成。
            
    -   创建临时表：开启后执行时会创建新的临时表存储生成的数据。
        
-   **沿线生成随机点**：沿线随机生成用于在一条线要素上，按一定规则或密度随机分布点要素，常用于模拟沿线事件、空间采样分析。选中线要素后单击![{CF866AC3-44D6-4944-AE73-D42778C2DBC3}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p966026.png)图标打开弹窗，配置信息后单击**执行**。
    
    -   最大数量：单条线上最多生成的数据量上限。
        
    -   缓冲半径：在线数据两侧一定范围内生成点的随机方向和距离，用于控制点分布的空间离散程度和覆盖范围。
        
    -   创建临时表：开启后执行时会创建新的临时表存储生成的数据。

**说明**

单击页面右侧的![{6D03057B-62BD-469C-ADD6-BE8D9354BF69}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p966098.png)和![{AC45A90B-D852-4722-89E8-AD111645262A}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p966099.png)图标，可选择展开或收起数据表面板。

**功能**

**说明**

单击**上传数据表**，上传地理数据文件，支持 CSV、GeoJSON、JSON 、TXT等文件格式。

单击![{52E4392E-AA7E-4CA6-A513-D72A010050A8}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p966091.png)图标，在弹窗中选择目标数据表，单击**执行**完成合并。当检测到字段冲突时，页面将出现提示，需用户确认后再次执行。

数据表删除

单击![{D7352962-B1C9-4263-A8AD-3B7240999CAC}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p966094.png)图标，在弹窗页面单击**确定**完成删除。

选中数据操作

支持在列头单选或批量选中数据，数据选中后支持定位、查看、复制和删除等操作。

-   **定位选中数据**：选中数据进行地图定位查看。
    
-   **查看选中数据**：选中数据进行GeoJSON格式数据预览。
    
-   **复制选中数据**：复制选中数据到剪贴板（不会生成新的数据）。
    
-   **删除选中数据：**选中数据后单击[地图面板](#section-vpe-mq0-hxi)中的![{7A72DE55-3736-4B86-B9D7-88BAAE19D6CC}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p966653.png)图标删除数据。
    

类型筛选

用于筛选表格和地图展示的数据几何类型。

数据字段

支持新增和删除数据字段操作。

-   **新增字段**：单击**新增字段列**，在弹窗中完成参数的配置后，单击**确认**完成新增。
    
    **说明**
    
    新增字段列后，可以在数据列表中进行内容编辑。
    
    -   字段名称：必填、格式支持英文、数字、汉字、下划线，不能以数字开头。
        
    -   字段类型：支持字符串和数字类型。
        
    -   是否默认化初始值：如果开启则展开默认值配置，否则默认值为空。
        
        -   字符串类型默认值参数：
            
            -   填充规则：可选唯一值和随机街道名称（模拟），默认唯一值。
                
            -   默认值：当选择唯一值的时候配置此参数。
                
            -   覆盖类型：该默认值规则覆盖的数据几何类型。
                
        -   数值类型默认值参数配置：
            
            -   填充规则：支持唯一值、值自增、随机范围，默认值自增。
                
            -   默认值：当选择唯一值的时候配置此参数。
                
            -   最小值、最大值：当填充规则选择随机范围时进行配置。
                
            -   覆盖类型：该默认值规则覆盖的数据几何类型。
                
-   **删除字段**：单击**删除字段列**，在弹窗中选择需要删除的字段名称，单击**确定**完成删除。
    

导出数据

支持表格数据导出为**GeoJSON格式**、**DataV 地图组件格式**、**SVG 格式**。

**说明**

导出SVG格式时会打开预览页面，支持鼠标滚轮缩放地图，单击**下载**按钮可将区域SVG数据保存至本地。

数据列表操作

数据列表支持数据定位和删除操作。

-   **数据定位**：单击数据列表中某一条数据后的**定位**按钮，进行地图定位查看。
    
-   **数据删除**：数据列表中某一条数据后的**删除**按钮，删除本条数据。
    

## **GeoAtlas智能助理**

GeoAtlas智能助理是一个高效、智能的地图工具调用专家，可以帮助用户快速完成地图相关的各种任务，比如找路线、查地点、模拟数据、数据分析等。深度集成Atlas MCP工具，能够自动调用路径规划、兴趣点（POI）搜索和数据生成等功能，借助大模型智能拆解需求和组合MCP 工具，极大的节省时间、提高效率。

**说明**

GeoAtlas智能助理在打开数据编辑器时默认开启，用户可通过AI助手开关按钮自由控制面板的显示/隐藏状态。

1.  单击页面底部的**登录阿里云账号**。
    
2.  激活MCP工具：可以在具体示例后单击**免费激活**，或在页面左下角工具箱中**点击激活**。进入MCP令牌管理页面。
    
    ![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2422990571/p966233.png)
    
3.  在MCP令牌管理页面中，单击**立即创建令牌并开始使用**生成令牌，生成后单击**激活**完成令牌激活。
