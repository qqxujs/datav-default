范围选择器是一款自定义快速选择并生成区域范围数据连接的小工具，本文介绍范围选择器的详细功能。

## 进入范围选择器

单击[DataV.GeoAtlas地理小工具系列](https://datav.aliyun.com/portal/school/atlas/area%5Fselector)，打开范围选择器工具页面。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7171567761/p574082.png)

## 范围选择器功能详解

| **功能**   | **说明**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 选择地图数据版本 | 单击页面右侧的****数据版本**下拉框，您可以自定义选择****areas\_v3**或****areas\_v2**两种地图数据版本。**说明** 当前范围选择器页面内的地图数据来源于高德开放平台。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| 选择区域范围   | 您可以通过两种方式选择地图中的区域范围，并展示选中区域范围的内容详情，包括展示当前选中区域范围的****地名**和****adcode**值。方法一：单击左侧地图处某个区域，即可在右侧选择区域范围处显示选中的区域范围并展示区域详情。**说明** 地图支持多次单击需要下钻的区域的功能，最小下钻范围至县级区域范围，单击下钻区域外空白区域，即可返回上一级区域。方法二：在右侧选择区域范围处，自定义搜索输入区域名称即可选中所需区域并展示区域详情数据。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| 选择数据格式   | 在选中地图中的区域范围后，可自定义选择所需的范围数据格式内容，可选数据格式内容包括如下几种：****JSON API**：单击链接右侧的![复制链接](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6847219361/p364288.jpg)图标，即可复制JSON数据的API链接。****GeoJSON API**：：单击链接右侧的![复制链接](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6847219361/p364288.jpg)图标，即可复制GeoJSON数据的API链接。![功能示例图](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3582590561/p433762.png)****其他类型**：如果您不需要系统自动生成API链接的数据格式，您可以选择****其他类型**中以具体GeoJSON数据内容或者范围数据图形的类型格式保存到本地。单击具体数据内容右侧的![复制链接](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6847219361/p364288.jpg)图标，即可复制所选区域的所有GeoJSON数据内容；单击数据内容最右侧的![下载图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6847219361/p364296.jpg)图标，即可下载所选区域的所有GeoJSON数据内容到本地。单击****SVG**右侧的![下载图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6847219361/p364296.jpg)图标，即可下载所选区域的图形格式内容到本地。**说明** 在选择需要的数据格式时，您可以勾选****包含子区域**，将区域范围内的子区域数据一起选中。 |
