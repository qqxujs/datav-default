  通过复制粘贴配置快速复用资产样式-DataV数据可视化-阿里云

通过复制并粘贴资产配置的功能，您可以快速将资产的配置参数复制到同类型的另一个资产内，以格式刷的效果高效完成多个同类型资产配置项参数之间的传递。本文详细介绍如何使用该功能。

## 背景信息

本功能仅适用于样式结构一致的同款资产，且只能复制粘贴资产的配置项样式内容，不包括资产的数据及交互配置。任何在配置项参数中涉及数据相关内容的都会被过滤，仅保留黏贴配置样式相关的内容。

## 操作步骤

1.  登录[DataV控制台](https://datav.aliyun.com/)。
2.  在**我的可视化**页面，单击您创建的可视化应用项目上方的**编辑**按钮。
3.  在画布编辑器页面，单击**图层**栏或画布中的某一个资产。
    
    **重要**
    
    在搜索资产配置之前，请确保您的可视化应用项目中已经添加了相应的资产，否则需要首先添加资产，详情请参见[添加资产](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/add-a-widget-1/#task-2375434)。
    
4.  单击当前资产**配置**面板左下角的**复制配置到剪贴板**，即可复制当前资产内的所有配置项参数内容。
    
    ![复制配置到剪贴板](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2647521361/p179967.png)
    
    **说明**
    
    当资产为BI分析模式的组件时，资产配置面板下方的**复制配置到剪贴板**功能，以![复制缩略图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2647521361/p325924.jpg)缩略图标样式呈现。
    
5.  在画布编辑器页面，单击**图层**栏或画布中的另一个同类型的资产。
    
6.  单击当前资产**配置**面板右下角的**粘贴组件样式**，即可将上面步骤中复制的同类型资产配置项内容，完全粘贴到当前资产的配置项参数内。
    
    ![粘贴配置项](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2647521361/p179974.png)
    
    **说明**
    
    当资产为BI分析模式的组件时，资产配置面板下方的**粘贴组件样式**功能，以![粘贴样式缩略图](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2647521361/p325927.jpg)缩略图标样式呈现。
