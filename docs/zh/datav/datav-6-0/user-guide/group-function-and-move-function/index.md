# 可视化应用功能管理 %{#87398}% 

本文档详细介绍了可视化应用管理的各项功能，涵盖项目管理和功能操作。这些功能帮助用户高效组织和维护可视化应用，提升工作效率。文档旨在帮助用户快速掌握功能的使用方法，优化管理流程，确保项目顺利进行。

## **进入可视化应用管理**

1. 登录[DataV控制台](https://datav.aliyun.com/)。
2. 登录控制台后，系统默认显示**我的可视化**页面。

## **项目管理**

项目管理功能是核心管理模块，支持用户高效组织、维护可视化应用项目。通过分组管理、排序管理和搜索管理等功能，用户可以清晰分类、快速定位和灵活调整项目，确保工作流程的高效性和条理性。该功能适用于团队协作、多项目管理和数据安全等多种场景，为用户提供全面的项目支持。

### **分组功能**

项目分组功能允许您对已创建的可视化应用进行分类和整理，使数据可视化项目更加有序且功能清晰。

1. 在**我的可视化**页面，单击**我的分组**右侧的**+**图标。
2. 在分组名称编辑框内，输入分组名，完成后单击空白处或者按**Enter**键确定。
3. 可选，您可以对已生成的分组进行重命名和删除操作。

**说明**  
删除分组后，该分组中的可视化应用将全部回归到未分组状态。
4. 移动鼠标到需要进行分组的可视化应用上，单击**移动**图标，将可视化应用直接拖动到左侧已创建好的分组栏中。在弹出的对话框中，单击**确定**。
5. 分组完成后，可单击分组名称，查看分组中包含的可视化应用。

### **搜索功能**

在**我的可视化**页面右侧的搜索框中，输入您要查找的可视化应用名称。DataV会自动进行匹配搜索，并展示符合搜索条件的可视化应用。

![大屏搜索页面](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1656728061/p9203.png)

**说明** 

搜索功能支持模糊匹配。

### **排序功能**

1. 在**我的可视化**页面，移动鼠标至页面最右侧的**按修改时间排序**（默认）上方。
2. 在下拉列表中单击选择一种可视化应用排序方式。  
![大屏排序页面](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5059822951/p9202.png)  
系统将根据您指定的排序条件，实时刷新并展示所有可视化应用的排列结果。

## **项目迁移**

旧版可视化应用可通过一键迁移功能升级到新版，确保数据完整性与功能兼容性，实现平滑过渡。

1. 在**我的可视化**页面中，鼠标移动到任一可视化应用区域，单击**迁移新版**。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5179265071/p758956.png)

**说明**  
移动端大屏暂不支持迁移。
2. 在**迁移大屏**确认窗口，单击**确定迁移**。
3. 等待系统迁移成功后，单击**查看大屏**即可进入迁移到新版数据大屏的编辑页。单击我的可视化页面右上角的**进入新版**，查看迁移后的大屏。

**说明**  
您也可以单击**编辑**进入可视化应用编辑页，单击页面右上角的**迁移新版**完成迁移。

## **项目拷屏**

可视化应用项目支持拷屏给他人和拷屏至工作空间操作，为用户提供灵活的项目管理方式和高效的团队协作体验。

### **拷屏给他人**

拷屏功能使您能够快速分享可视化应用的当前视图，便于与他人协作，共同推进应用的开发和优化。

**重要** 
* 拷贝过去的可视化应用包含完整的数据配置，为避免数据泄露，在拷贝前请仔细核对用户识别码。
* 拷屏功能只适用于**企业版**及以上版本。

1. 在**我的可视化**页面中，鼠标移动到任一可视化应用区域内，单击**拷屏**图标。
2. 在**拷贝项目给他人**对话框中，输入对方的用户识别码（区分大小写）。  
获取用户识别码：在**我的可视化**页面，将鼠标移至用户名称上，单击即可复制用户识别码。
3. 单击**拷贝**，拷贝当前项目给其他用户。  
拷屏完成后，您可以在**拷贝项目给他人**对话框中的**拷屏历史**中，查看对方的用户识别码和拷屏时间，帮助您及时了解拷屏动态。

### **拷屏到工作空间**

您可以将同项目的可视化应用拷贝到同一个工作空间中，方便进行备份管理和权限适配。

1. 在**我的可视化**页面中，选择一个可视化应用，单击**拷屏到工作空间**图标。
2. 在弹出的窗口中选择需要移动的工作空间，单击**确定**。
3. 等待拷屏成功后，返回目标工作空间查看已拷贝的可视化应用。

## **项目操作**

可视化应用项目支持编辑、删除、复制等多项功能操作，方便用户灵活管理和调整项目内容。

### **编辑功能**

1. 在**我的可视化**页面中，鼠标移动到任一可视化应用区域，单击**编辑**。
2. 在可视化应用编辑页面，修改组件的布局和配置，或者添加、删除组件，完成可视化应用的编辑。
3. 可选，编辑完成后，可预览或发布可视化应用，查看完成效果。

### **复制功能**

通过可视化应用的复制功能，您可以在现有应用的基础上快速创建类似的可视化应用，或将其用作开发和测试环境进行修改更新，而不会影响正在运行的在线生产应用。

1. 在**我的可视化**页面中，鼠标移动到任一可视化应用区域，单击**复制**。
2. 复制成功后，系统会自动在应用列表前生成一个名为`xxx_copy`的可视化应用。您可以修改该应用的名称，并在此基础上调整布局和配置。

### **删除功能**

删除不需要的可视化应用，可以释放可视化应用的空间与配额。

**重要** 

可视化应用删除后不可恢复，请谨慎操作。

在**我的可视化**页面中，选择一个需要被删除可视化应用。 

* 可视化应用已发布

  1. 单击可视化应用右上角的**发布**图标，在**发布**对话框中，关闭**已发布**开关，并单击**确定**。
  2. 再次单击**删除**图标，在二次确认删除对话框中，单击**确定**即可删除该可视化应用。
* 可视化应用未发布：直接单击**删除**图标，在二次确认删除对话框中，单击右下角的**确定**即可删除该可视化应用。

### **重命名功能**

重新命名一个可视化应用，帮助您快速修改可视化应用的名称，便于搜索与管理。

1. **我的可视化**页面中，单击对应可视化应用的名称，进入编辑状态，输入新的名称。  

**说明**  
可视化应用的名称可以重复。
2. 单击空白处或按**Enter**键，即可完成可视化应用名称的修改。  
可视化应用名称修改成功后，您就可以完成使用名称搜索可视化应用，按名称对可视化应用进行排序等操作。
