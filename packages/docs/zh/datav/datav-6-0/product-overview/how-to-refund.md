# 退款说明 %{#424670}% 

如果您购买DataV数据可视化后需要退订，您可以通过**资源退订**对实例进行退订操作。本文为您介绍如何进行DataV数据可视化退订操作。

## 退订概念

DataV数据可视化支持的退订类型：**五天无理由退订**和**非全额退订**。

如果您购买DataV数据可视化五天以内或超过五天且有正当理由需要退订，可以按照以下步骤操作。

**说明** 

DataV目前已支持自助退订和人工提交工单退订两种方式。

## 自助退订操作流程

**说明** 

在实例释放后，已保存的可视化项目及其他生产内容我们将会为您**保留15天时间**，期间重新购买产品可继续您之前的工作。

1. 进入[费用与成本](https://billing-cost.console.aliyun.com/home)。
2. 在左侧导航栏，单击**订购订单** \> **资源退订**。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7622891271/p826084.png)
3. 进入**资源退订**页面，选择**退订资源**选项卡，并选择**非全额退订**类型，搜索保有的实例。选择实例并单击**退订资源**。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7622891271/p826097.png)
4. 按您的实际情况选择并输入退订原因，阅读**退订规则**并勾选确认，单击**立即退订**。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7622891271/p826118.png)
5. 确认数据备份完成后，单击**确定**。  
![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7622891271/p826121.png)
6. 退订申请成功，等待处理完毕。![退订成功](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7862486661/p508765.png)

## 人工退订操作流程

针对不支持自助退订的云产品及实物类商品，可[提交工单](https://selfservice.console.aliyun.com/ticket/createIndex?spm=a2c4g.11186623.0.0.49126347e8gHcm)人工审核退款。

1. 提交工单，选择**财务类** \> **退订咨询**，说明退订原因、退订产品、退订实例等。
2. 由工程师审核是否符合退订条件，审核通过后工程师协助客户完成退订。确认退订后，实例立即释放，款项将于2-3个工作日到账。
