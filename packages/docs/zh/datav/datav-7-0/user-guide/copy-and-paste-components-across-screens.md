 

通过跨屏复制并粘贴组件的功能，您可以快速将已在其他项目中配置好的组件复制到新的项目中。本文详细介绍如何使用该功能。

## **背景信息**

通过跨屏复制并粘贴组件的功能，您可以快速将已在其他项目中配置好的组件复制到新的项目中。本文详细介绍如何使用该功能。

## **操作步骤**

1.  登录[DataV控制台](https://datav.aliyun.com/v/console/)。
    
2.  在**工作台**页面，选择任意一个数据看板，进入**画布编辑器**页面。
    
3.  在画布编辑器进行组件选择，支持组件多选。
    
    **说明**
    
    -   选中：鼠标点击
        
    -   多选：Ctrl | ⌘ + 鼠标点击
        
    
4.  对于选好的组件，使用键盘输入 C**trl | ⌘****+C** 进行复制，提示“已复制组件及交互，支持跨项目粘贴”。
    
5.  切换到需被粘贴的项目画布编辑器， 使用键盘输入C**trl | ⌘****+V** 进行粘贴，在画布编辑器中显示粘贴的组件信息。
    
    **说明**
    
    -   跨屏复制组件、变量后，保留内部的交互连线，只删除复制范围外的连线
        
    -   保留复制内容到目标屏内同名全局变量的连线。
