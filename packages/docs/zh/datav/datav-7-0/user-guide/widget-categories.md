 

本文介绍DataV服务中组件的类型和示例。

目前 DataV官方组件库支持的组件有以下几种：

-   图表
    
-   地图
    
-   信息
    
-   表格
    
-   控件
    
-   媒体
    
-   其他
    
-   区块
    
-   城市
    
-   我的
    

## 图表

您可以在**图表**组件中，选择您需要的图表，包括柱状图、折线图、饼环图、散点图和其他等。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7865630961/p698395.png)

## 地图

您可以在**地图**组件中，选择您需要的地图，包括基础平面地图、3D平面地图和3D地球。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7865630961/p698397.png)

## 信息

您可以在**信息**组件中，选择您需要的组件，包括通用标题、跑马灯、数据翻牌器等。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7865630961/p698398.png)

## 表格

您可以在**表格**组件中，选择您需要的组件，包括轮播列表柱状图、进度条表格等。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7865630961/p698399.png)

## 控件

您可以在**控件**组件中，选择您需要的组件，包括时间轴、Tab列表等。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7865630961/p698401.png)

## 媒体

您可以在**媒体**组件中，选择您需要的组件，包括视频、音频等。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7865630961/p698402.png)

## 其他

您可以在**其他**组件中，选择您需要的辅助图形，包括矩形树图、抽奖轮播图和装饰等。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7865630961/p698403.png)

## 区块

您可以通过**区块**，添加自定义的区块组件。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7865630961/p698404.png)

## 城市

您可以通过**城市**，添加自定义的三维城市组件。仅限**专业版**及以上版本支持。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7865630961/p698405.png)

## 我的

您可以通过**我的**，查看组件包中的自定义开发组件。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7865630961/p698406.png)
