# 三维城市构建器项目管理 %{#429600}% 

三维城市构建器是一款专为创建和展示三维城市模型设计的工具，旨在为用户提供强大的城市建模与管理功能。本文将详细解析其核心功能，包括项目管理、项目创建与项目编辑，帮助用户高效组织、执行和协作，从而顺利完成三维城市的构建与管理任务。

## **进入三维城市构建器**

1. 登录[DataV控制台](https://datav.aliyun.com/)。
2. 点击页签中的**三维城市构建器**，进入三维城市构建器的主页面。

## **项目管理**

三维城市构建器中的项目管理功能是用户高效组织、管理和维护三维城市模型项目的核心模块。通过分组管理、排序管理和搜索管理等功能，用户可以清晰分类、快速定位和灵活调整项目，确保工作流程的高效性和条理性。该功能适用于团队协作、多项目管理和数据安全等多种场景，为用户提供全面的项目支持。

### **分组管理**

项目分组管理功能可以将已经创建的三维城市构建器项目进行分组归类，使得三维城市构建器项目变得有序且功能性一目了然。

#### **创建与管理分组**

1. 在**三维城市构建器**页面，单击**我的分组**右侧的**+**按钮。
2. 在分组名称编辑框内，输入分组名，完成后单击空白处或者按**Enter**键确定。
3. 您可对已创建的分组进行重命名或删除操作。删除分组后，该分组内的所有三维城市构建器项目将自动回归至未分组状态。![重命名/删除分组](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3200468461/p412784.png)

#### **移动项目至分组**

1. 将鼠标悬停于目标三维城市构建器项目上，单击右下角的![{A54838A2-C0BA-468D-B5D8-7ED29766C8F3}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2869481471/p925144.png)图标，从下拉菜单中选择**移动**选项。
2. 在弹出的**选择分组**对话框中，可以选择将项目添加到**已有分组**或**新建分组**，在下方选择一个已有分组名称或输入新建分组的名称，单击对话框右下角的**完成**即可完成项目的分组。![{3FABA006-DD1F-40BB-82F3-0E95A081B828}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2869481471/p925147.png)  
项目选择分组完成后，可单击左侧分组的名称，查看分组中包含的三维城市构建器项目。![{C7AA5B34-68D5-439D-901A-DE40FE60131F}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2869481471/p925149.png)

### **排序管理**

排序管理是三维城市构建器中项目管理功能的重要组成部分，旨在帮助用户根据特定规则对项目或分组进行排序，从而更高效地组织和查找目标内容。

1. 在**三维城市构建器**页面，移动鼠标至页面最右侧的**按修改时间排序**（默认）上方。
2. 在下拉列表中，单击选择一种三维城市构建器项目的排序方式。![{674B7961-4777-456E-9C8C-8008033A3463}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2869481471/p925155.png)

三维城市构建器项目排序成功后，即可看到所有三维城市构建器项目按照您选择的方式进行排序。

### **搜索管理**

搜索管理是三维城市构建器中项目管理功能的重要组成部分，旨在帮助用户快速定位和查找特定的项目、分组或内容。

在三维城市构建器页面右侧的搜索框中，输入您要查找的三维城市构建器项目名称。DataV会自动进行匹配搜索，并展示符合搜索条件的三维城市构建器项目。![搜索项目](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9400468461/p412825.png)

**说明** 

搜索功能支持模糊匹配。

### **项目版本历史管理**

三维城市构建器项目的版本历史功能仅在项目发布后可用。每次发布都会为当前项目内容生成一个快照，并分配一个新的版本号。未发布过的项目不会生成历史版本信息。

1. 在三维城市构建器页面中，单击某个已发布项目上的**查看历史**。
2. 在右侧弹出的历史版本面板中，您可以查看当前项目的所有版本，包括项目的**版本号**、**发布时间**、**说明**、**状态**和**操作**等信息。![{F0EEC869-B8A7-4FDD-BFD7-E038FEDA949E}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2869481471/p925220.png)

|    **项目版本信息参数**    |    **说明**    |
|    ----    |    ----    |
|    **版本号**    |    一个项目可对应多个版本号。最新使用的版本号等于已上线最大版本号。您可以在画布编辑器中使用项目，并在右侧配置面板中单击**更新版本**，将项目升级到最新版本后继续使用。    |
|    **发布时间**    |    每一个版本项目的发布详细时间。    |
|    **说明**    |    发布项目时自定义设置的版本情况说明。    |
|    **状态**    |    项目版本的上下线状态，包括**已上线**和**已下线**两种。    |
|    **操作**    |    针对不同状态下的历史版本的项目，可对项目进行**上线**或**下线**的操作。**上线**：单击一个**未上线**状态下项目版本右侧的![上线](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3664344261/p282361.jpg)图标，在弹出的**上线版本**对话框中，单击**确定**即可上线该版本项目内容。![上线区块版本](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3664344261/p282363.jpg)**下线**：单击一个**已上线**状态下项目版本右侧的![下线图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3664344261/p282362.jpg)图标，在弹出的**下线版本**对话框中，单击**确定**即可下线该版本项目内容。![下架版本](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3664344261/p259310.jpg)    |

**说明**  
默认情况下，一个项目被发布后都处于**已上线**状态。项目上线后可以在可视化应用画布编辑器中，单击**更新版本**按键，将项目升级到最新版本后继续使用。具体使用项目的方法请参见[查看并添加使用项目](/zh/datav/datav-6-0/user-guide/publish-and-use-dashboards#section-9rd-8jq-d1t)。
3. 单击历史版本面板左上角的**收起**，即可关闭历史版本面板。

## **创建项目**

三维城市构建器项目支持通过**官方基底数据**和**空间构建数据**两种方式，灵活添加多个城市基底子组件和数据映射子组件。如需了解更多操作细节，请参见[创建三维城市场景项目](/zh/datav/datav-6-0/user-guide/create-3d-city-projects)。

## **项目操作**

三维城市构建器项目支持编辑、删除、复制等多种功能操作，方便用户灵活管理和调整项目内容。

### **编辑项目**

1. 在**三维城市构建器**页面中，选择一个三维城市构建器项目，单击**编辑**图标。
2. 在三维城市构建器的**场景编辑器**页面中，您可以修改项目场景内容、配置场景样式、添加数据组件，并设置关键帧与分镜，从而完成三维城市项目的场景编辑。有关场景编辑器的详细功能说明，请参见[场景编辑器管理](/zh/datav/datav-6-0/user-guide/scene-editor-management-of-professional-edition/)。此外，您还可以配置项目的数据、事件和动作接口，或切换到**蓝图编辑器**中设计项目的交互效果。如需了解蓝图交互功能的详细信息，请参考[蓝图编辑器配置说明](/zh/datav/datav-6-0/user-guide/blueprint-editor-configuration#concept-2193562)**。**

### **重命名项目**

在**三维城市构建器页**面中，单击目标项目的名称，即可进入编辑状态，对项目进行重命名操作。

项目名称修改成功后，您就可以完成使用名称搜索三维城市构建器项目，按名称对三维城市构建器项目进行排序等操作。

**说明** 

三维城市构建器项目的名称可以重复。

### **删除项目**

**重要** 

三维城市构建器项目删除后不可恢复，请谨慎操作。

将鼠标悬停于需要删除的三维城市构建器项目上，单击右下角的![{A54838A2-C0BA-468D-B5D8-7ED29766C8F3}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2869481471/p925144.png)图标打开下拉菜单，选择**删除**选项，在弹出的二次确认对话框中，点击**确定**即可完成项目删除。

### **复制项目**

将鼠标悬停于需要复制的三维城市构建器项目上，单击右下角的![{A54838A2-C0BA-468D-B5D8-7ED29766C8F3}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2869481471/p925144.png)图标打开下拉菜单，选择**复制**选项，

复制成功后，系统将在项目列表顶部自动生成一个名为**xxx\_copy**的项目副本。您可对该副本进行重命名，并基于此项目修改场景内容、调整子组件数据等，快速创建新的三维城市构建器项目。

### **预览项目**

* 方式一：将鼠标悬停于需要预览的三维城市构建器项目上，单击右上角的**预览**图标，即可直接预览项目内容。
* 方式二：选择一个三维城市构建器项目，单击**编辑**图标，在项目场景编辑器中，单击页面工具栏右上角的![{B8C9D596-C2C1-4D46-87C2-84E58ECFDC13}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2869481471/p925212.png)图标，加载后即可直接预览三维城市构建器项目内容。
