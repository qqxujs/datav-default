本文介绍可视应用编辑器中的用户账号的基本内容。包括登录账号、版本信息、用户识别码等。

| **名称** | **说明**                                                                                                                                 |
| ------ | -------------------------------------------------------------------------------------------------------------------------------------- |
| 登录账号   | 展示当前用户登录的账号名称。                                                                                                                         |
| 版本信息   | 展示当前环境的版本信息，包括但不限于企业版、专业版和尊享版。                                                                                                         |
| 用户识别码  | 展示当前登录用户的个人识别码，可以根据当前识别码接收来自于其他使用者的可视内容分享和拷屏。                                                                                          |
| 产品新购   | 单击![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8892522961/p707421.png)图标，跳转至**产品通用售卖**页面，您可以购买其他本产品的其他版本。 |
| 退出登录   | 单击![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8892522961/p707423.png)图标，退出当前用户。                          |
