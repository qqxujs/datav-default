 

尊敬的 DataV-Board 用户：

为持续提升产品安全性并优化用户体验，阿里云 DataV-Board 将对现有授权方式进行调整。

经评估，基于 RAM 授权AdministratorAccess和AliyunDataVFullAccess的登录方式存在潜在安全风险，不符合当前企业级数据可视化服务的安全规范要求。为保障您的数据与系统安全，我们计划于 2025年6月18日起全面关闭RAM中AdministratorAccess和AliyunDataVFullAccess授权进入DataV任意工作空间开发编辑项目的高风险权限。

## 关闭RAM授权方式**变更影响**

-   自 2025年6月18日 起，DataV-Board 将不再支持通过 RAM中AdministratorAccess和AliyunDataVFullAccess的授权进入DataV任意工作空间开发编辑项目。
    
-   影响范围：当前使用RAM中AdministratorAccess和AliyunDataVFullAccess权限进入企业版、专业版、尊享版用户。
    

## **替代方案与操作指引**

请受影响的企业版、专业版、尊享版通过 RAM中AdministratorAccess和AliyunDataVFullAccess 的子账号用户立即通过以下链接完成工作空间访问权限更新即可正常使用：[https://datav.aliyun.com/v/workspace](https://datav.aliyun.com/v/workspace)
