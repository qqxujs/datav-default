本文介绍蓝图编辑器中的快捷功能，包括编组收纳、节点说明和快速连线等。

## **编组收纳**

当蓝图中节点间的逻辑较为复杂时，您可以将相似逻辑的节点进行编组，系统将自动生成一个编组框，方便您的查看和使用。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3232680861/p624312.png)您可以使用以下编组功能。

* **添加编组**：右键单击画布中的一个或多个节点，选择**添加编组**，可生成一个编组框。您可以调整编组框的大小，后续只需要拖动编组框即可同时移动所有编组框内的节点。  
![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5225704961/p716446.png)![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5225704961/p716448.png)
* **删除选中**：当您不需要编组功能时，右键单击编组框，单击**删除选中**即可。

**重要**  
删除选中后不可恢复，请谨慎解除。  
![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5225704961/p716449.png)
* **收起编组**：当您将当前编组框内的节点配置完成后，可以单击编组左上角的![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5225704961/p716453.png)图标收起编组，减少视觉上的繁杂感。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5225704961/p716457.png)![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5225704961/p716459.png)
* **展开编组**：当您需要重新编辑当前编组框内的节点时，可以单击编组左上角的![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5225704961/p716462.png)图标展开编组。
* **编辑编组名称**：单击当前编组框左上角的![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3627982761/p543550.png)图标，自定义修改编组框的名称。![c3155a62548702be6014130d155fd3ce](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3627982761/p543552.png)

## **节点说明**

**说明** 

添加注释功能为7.0版本产品新增功能，请根据[新版控制台](https://datav.aliyun.com/v/console/)环境进行操作。

您可以在任意类型节点上右键单击**添加说明**，给当前节点添加注释，帮助理解节点间的逻辑关系。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3627982761/p543481.png)

## **快速连线**

**说明** 

快速连线功能为7.0版本产品新增功能，请根据[新版控制台](https://datav.aliyun.com/v/console/)环境进行操作。

您可以左键单击选中两个锚点，使用ctrl+l/cmd+l，即可在两个锚点之间自动添加一条连线。连线的其他操作请参见[连线操作](https://help.aliyun.com/zh/datav/datav-7-0/user-guide/blueprint-editor-features-of-datav-7#section-w05-9to-kmy)。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5225704961/p716481.png)
