  使用疫情态势管控模板创建可视化应用-DataV数据可视化-阿里云

本文为您介绍使用疫情态势管控模板创建可视化应用的方法。

## 操作步骤

1.  登录[DataV控制台](https://datav.aliyun.com/)。
2.  在我的可视化页面中，单击PC端创建。
    
    ![新建可视化](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2124968461/p10314.png)
    
3.  您可以浏览如下图所示的多款免费疫情模板可视化应用的效果，选择其中一款，单击创建项目。
    
    **说明** 您还可以创建一个空白画布并自行设计一款疫情可视化应用，具体请参见[创建可视化应用](https://help.aliyun.com/zh/datav/use-a-blank-template-to-create-a-project#task-2462286 "本文为您介绍使用空白模板创建可视化应用的方法。")。
    
    ![疫情模板](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0256822951/p81066.png)
    
4.  以疫情态势管控大屏模板为例，在创建数据大屏对话框中，输入可视化应用名称，并设置应用的分组。
5.  单击创建。
    
    应用创建成功后会跳转到画布编辑器页面，即可看到一款满足疫情态势管控功能且设计精良的模板。![进入疫情模板编辑器](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0256822951/p81068.png)
