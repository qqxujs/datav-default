## 登录宜搭平台

操作步骤：

1. 登录[宜搭平台](https://www.aliwork.com/home/)，在宜搭官网首页右上角单击**登录**。![宜搭首页](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1920660461/p375175.png)

**说明**  
  * 如果您还没有宜搭账号，请在宜搭官网首页单击右上角注册，输入您的手机号注册钉钉账号即可登录宜搭。![注册钉钉](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1920660461/p376968.png)
  * 如果您已经有钉钉账号，可以直接登录宜搭。
2. 在**登录宜搭**页面，单击**立即登录**。![登录宜搭](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1920660461/p375184.png)
3. 选择所在的组织：**钉钉宜搭**。![选择加入的组织](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1920660461/p375190.png)

**说明**  
  * 您可以在登录时选择您所加入的组织。
  * 您也可以登录后，在宜搭工作台的首页右上角切换组织。![切换组织](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1920660461/p375270.png)

## 创建模板大屏

操作步骤：

1. 在**创建应用**模块，选择**创建空白应用**，并单击**开始创建**。![创建应用](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1920660461/p375278.png)
2. 在**创建应用**对话框中，输入**应用名称**，并单击**确定**。![创建应用](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1920660461/p375283.png)

**说明**  
您也可以通过宜搭首页**我的应用**页签下**创建空白应用**，也可以查看和编辑已有的应用。![我的应用](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1920660461/p376115.png)
3. 在**页面管理**页面，单击**新建大屏**。![新建大屏](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1920660461/p375966.png)
4. 在**新建大屏**对话框中，输入**页面名称**，同时您也可以**选择分组**，并单击**确定**。![新建大屏](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2920660461/p376995.png)
5. 在**模板列表**中，选择一个大屏模板，单击**就选它**后，即可创建一个模板大屏。本文以**招投标行业大屏**为例介绍大屏的相关配置。
6. 创建完的大屏效果展示。

## 配置组件数据源

本文以配置大屏内**梯形柱状图**组件数据源为例。

操作步骤：

1. 在创建完成的模板大屏中，单击选择**交易趋势**梯形柱状图组件。
2. 在组件右侧配置面板中，选择**数据**面板，在下方**数据响应结果**模块，单击**配置数据源**。![配置数据源](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2920660461/p376037.png)
3. 在**设置数据源**面板，数据源类型下拉列表中选择数据源，本文以**宜搭数据源**为例，单击**进入配置**。![选择数据源](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6294872561/p439910.png)
4. 在**图表数据配置**面板，选择一个您已经自定义配置完成的数据集内容，并通过拖拽方式将数据集内左侧的字段添加到中间**图表字段映射**中。同时，您可以在右侧**图表预览**区域中查看您图表配置数据后的样式，并单击**确定**。![配置数据源](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2920660461/p376113.png)
5. 您可以在大屏编辑页面，查看配置数据后的组件效果。

## 配置组件样式

本文以配置大屏内**通用标题**组件样式为例。

操作步骤：

1. 在大屏编辑页面，以**招投标行业可视化**模板大屏为例，单击您需要修改的**通用标题**组件。
2. 在组件右侧配置面板，将**标题名**处修改为**宜搭产品销售可视化大屏**。![配置标题](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3920660461/p376960.png)
3. 在大屏编辑页面，可以查看修改标题后的展示效果。
4. 您可以根据上述步骤方法，自定义修改各种组件的样式配置项和数据内容，并随时查看组件修改后的效果。

## 配置蓝图编辑器功能

如果您需要通过可视化连线定义多个组件之间的交互关系，您可以使用蓝图编辑器功能。具体操作步骤如下：

1. 在画布编辑器，左侧**图层**栏选中需要交互的组件，本文以**通用标题**和**Tab列表**组件为例，分别右键单击**导出到蓝图编辑器**。
2. 导出成功后，单击左上角![蓝图编辑器按钮](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7959772561/p439871.png)按钮，进入蓝图编辑器。![蓝图编辑器](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7959772561/p439876.png)
3. 在蓝图编辑器页面，将**导入节点**面板中的Tab列表和通用标题，拖至画布上并连线。具体的配置方法请参见[配置蓝图编辑器](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/what-is-blueprint-editor)。  
![蓝图编辑器连线](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7959772561/p439881.gif)
4. 配置完成后，您可以在画布编辑器右上角单击预览按钮，预览设置的效果。
