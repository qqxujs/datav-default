# 规格说明 %{#141426}% 

DataV提供了**企业版**、**专业版**和**尊享版**三种版本供您选择。下表介绍了部分功能在对应版本中的支持情况，详情请参见[版本特性](https://www.aliyun.com/activity/intelligent/datavedition)。

|    **功能**    |    **企业版**    |    **专业版**    |    **尊享版**    |
|    ----    |    ----    |    ----    |    ----    |
|    **静态文件导入**    |    支持    |    支持    |    支持    |
|    **常见数据库接入**    |    支持    |    支持    |    支持    |
|    **可用工作空间**    |    2个    |    5个    |    100个    |
|    **可创建项目数**    |    20个    |    40个    |    1000个    |
|    **模板创建**    |    基础模板（不含三维模板）    |    所有模板    |    所有模板    |
|    **组件收藏数**    |    20个    |    40个    |    1000个    |
|    **2D平面地图组件**    |    支持    |    支持    |    支持    |
|    **3D平面地图组件**    |    不支持    |    支持    |    支持    |
|    **三维模型查看器**    |    支持    |    支持    |    支持    |
|    **设计资产管理**    |    支持5GB    |    支持10GB    |    支持99GB    |
|    **自定义组件数量**    |    3个    |    无限制    |    无限制    |
|    **低代码组件开发**    |    不支持    |    支持    |    支持    |
|    **高级交互组件**    |    不支持    |    支持    |    支持    |
|    **二次开发**    |    不支持    |    不支持    |    支持    |
|    **本地部署**    |    不支持    |    不支持    |    支持    |
|    **可创建场景数量**    |    不支持    |    10个    |    50个    |
|    **可创建场景大小**    |    不支持    |    49平方公里    |    无限制    |
|    **地图服务接入**    |    不支持    |    不支持    |    支持    |

**说明** 
* 项目：表格中创建的项目是指可创建的可视化大屏，购买不同的版本可创建不同数量的项目，如果需要创建更多的项目，可以**购买扩展包**实现。如您在企业版账号下已经创建了20个项目，还果您想继续创建第21个，则需要购买扩展包。
* 工作空间：一个工作空间相当于一个新的控制台，可以分给不同的开发者分别创建自己的项目，不同的工作空间数据是隔离的，可以配置自己的数据源。但所有工作空间创建的项目总数不能超过购买版本的最大数量。如A开发者创建了15个项目，那么B开发者没有购买扩展包的前提下最多只能创建5个项目，但A、B两个开发者开发的项目之间数据互不影响。
* 可视化组件：可视化大屏是由多个组件构成，您可以通过对组件进行配置来完成您的可视化大屏设计，同时DataV产品也开通了自定义组件开发功能，您可以根据自己的需求进行组件开发。
* 本地部署：也叫私有化部署，是同一个概念，只有尊享版支持该功能。在实现上您可以将DataV开发的页面整个打包下载，安装部署到企业内网中运行，整个部署均在线下完成，数据完全隔离，保障了数据的安全性，同时尊享版共提供100台服务器的部署授权（不包含硬件），保障产品更好地在企业内网中运行。
* 二次开发：尊享版支持编写JavaScript代码（hook脚本）的方式进行可视化页面的开发，实现项目内各种复杂交互效果的实现，拥有比蓝图编辑器更高的自由度和灵活度，可以方便开发人员自己随意定制心仪的项目，二次开发的详细内容请参见[DataV二次开发教程使用说明](/zh/datav/datav-6-0/user-guide/datav-custom-development-instructions)。
