# 项目编辑器页面管理 %{#290826}% 

项目编辑器内可分为四个区域，包括空间实体、通用元素、数据详情和项目地图空间范围。本文介绍项目编辑器页面中各个功能的使用方法。

## 空间实体

空间实体包括**环境**、**建筑**、**三维瓦片**和**地理服务**等。

* **环境**：包括**水体**、**绿地**、**道路**和**遥感影像-道路**。
* **建筑**：包括**矢量建筑**、**精模建筑**和**BIM建筑**。
* **三维瓦片**：包括**倾斜摄影**、**BIM**、**管网**、**点云**、**手工模型**和**混合类型**。
* **地理服务**：包括**XYZ**、**WMTS**、**WMS**、**ArcGISRest**和**SuperMapRest**。

在**空间实体**区域内可以实现的功能有以下几种：

|    **功能**    |    **说明**    |
|    ----    |    ----    |
|    **添加基底数据**    |    单击空间实体区域页面上方的**添加基底数据**，即可打开**添加基底数据**对话框，您可以参见[添加项目基底数据](/zh/datav/add-project-base-data-for-datav-6#concept-2043131)，在原有空间区域数据的基础上添加一个全新的空间范围基底数据。![添加基底数据入口](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242520.png)**重要** 当您添加基底数据时，如果重新添加了同一个区域的空间实体数据内容，所有数据会重复叠加显示。    |
|    **展开实体列表**    |    单击实体父节点或一级子类节点左侧的![加号](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242554.jpg)或![减号](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242555.jpg)图标，即可展开或折叠实体列表中具体类别实体的列表样式。    |
|    **新增实体**    |    您可以单击某类实体右侧的![加号](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242526.jpg)图标，在对话框内选择一种实体子类后，即可添加某个具体子类型的实体数据，添加后的实体会排列显示在当前类别实体列表的最下方。![新增实体方法一](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242531.png)您也可以直接单击某类实体子类型节点右侧的![加号](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242526.jpg)图标，即可添加对应子类型的实体数据，添加后的实体会排列显示在当前类别实体列表的最下方。![新增实体方法二](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242532.png)    |
|    **编辑实体**    |    单击具体列表中已有实体右侧的![编辑](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242535.jpg)图标，可在左侧实体名称输入框内自定义修改实体名称，修改实体名称后鼠标单击空白处或按**Enter**键，即可完成实体名称的修改。    |
|    **删除实体**    |    单击具体列表中已有实体右侧的![删除](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242538.jpg)图标，在删除对话框内，单击**OK**即可删除当前实体内容。    |
|    **隐藏或显示实体**    |    单击具体列表中已有实体右侧的![隐藏](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242540.jpg)图标，即可在右侧地图区域隐藏当前实体对应的内容。您可以再次单击当前实体右侧的![显示](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242543.jpg)图标，即可显示该实体内容。![实体显隐样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9925887161/p242545.png)    |

## 通用元素

通用元素区域共有三种元素内容，分别是**点**、**线**和**面**元素，您可以在地图空间数据范围内自定义配置元素内容。

**说明** 

默认状态下，通用元素页面中没有任何已有元素，需要自定义配置。

在通用元素区域中可以实现以下功能：

|    **功能**    |    **说明**    |
|    ----    |    ----    |
|    **添加数据**    |    单击通用元素区域上方的**添加数据**，即可打开**添加数据**对话框，您可以在对话框内，单击选择您需要的数据，并单击对话框右下角的**确定**即可完成数据添加。![添加数据对话框](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7353360261/p262471.png)完成添加后的数据，会加载排列在对应数据元素类型列表内。![排列显示在面元素中](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7353360261/p262467.png)    |
|    **新增元素**    |    单击通用元素区域中某个类型元素右侧的![加号](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242526.jpg)图标，即可打开**新建数据集**对话框，在对话框内，您可以配置如下信息。![新增数据集](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5286522561/p438127.png)配置数据集名称：您可以自定义输入数据集名称，为当前类型的元素命名。查看并配置字段：您可以在对话框中查看到当前新建元素所对应的字段内容，不同的类型元素字段都不相同。单击对话框中的**添加一行数据**，即可添加一行字段。您可以自定义配置新增字段的名称、字段类型和字段描述。![新增字段](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7353360261/p262395.png)保存数据集：单击**新建数据集**对话框右下角的**确定**即可完成元素数据集的添加。    |
|    **配置元素字段**    |    单击某个元素右侧的![字段配置入口](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7353360261/p262422.jpg)图标，即可打开**字段配置**对话框，系统字段不支持修改，您可以对添加的字段进行编辑或删除的操作，也可新增一行字段数据。    |
|    **编辑元素**    |    单击某个元素右侧的![编辑](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242535.jpg)图标，可在左侧元素名称输入框内自定义修改元素名称，修改元素名称后鼠标单击空白处或按**Enter**键，即可完成元素名称的修改。    |
|    **删除元素**    |    单击某个元素右侧的![删除](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242538.jpg)图标，在删除对话框内，单击**OK**即可删除当前元素内容。    |
|    **隐藏和显示元素**    |    单击某个元素右侧的![隐藏](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242540.jpg)或![显示](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242543.jpg)图标，即可隐藏或显示该元素内容。    |
|    **展开元素目录**    |    当在某种元素中添加了一个或多个元素后，您可以单击元素类型父节点左侧的![加号](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242554.jpg)或![减号](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8925887161/p242555.jpg)图标，即可展开或收起某一类元素目录中具体元素的列表样式。    |

## 数据详情

数据详情页面指的是在您选中具体某个实体或元素数据后打开的页面。数据页面中包括对应实体或者元素内所有的数据信息内容。

在数据详情页面内可以实现的功能有以下几种：

|    **功能**    |    **说明**    |
|    ----    |    ----    |
|    **查看并修改数据属性**    |    单击数据页面内任意一个具体数据，在右侧会弹出当前数据的详情页面，页面内可以查看不同类型数据的不同属性，您可以自定义设置每一个数据的属性字段参数值。![查看并修改属性](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5286522561/p438136.png)**说明** 精模建筑和地标建筑实体数据除了属性参数外还包括模型参数内容，详情请参见下方**修改地标及精模建筑数据模型参数**。    |
|    **添加数据**    |    本操作以添加一个实体数据为例（地标建筑和精模建筑数据除外）。单击选中一个实体数据，在展开的面板中单击页面上方的**添加**按钮。![添加数据](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8353360261/p263846.png)在弹出的**添加数据**对话框中选择一个已有的适配类型数据，单击对话框右下角的**下一步**即可进入映射选中数据字段页面。如果此处无待添加的数据，请参见[上传数据](/zh/datav/datav-6-0/user-guide/upload-data-for-datav-6#task-2069055)章节上传所需数据文件。![添加数据](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9068242561/p438637.png)在映射字段属性页面，单击属性下拉框，选择映射数据样例中某个已有的字段内容，配置映射属性字段，完成后单击**下一步**即可进入预览数据页面。![映射字段属性](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9068242561/p438640.png)在预览数据页面查看数据解析成功后，单击右下角**添加全部数据**即可添加所有新解析后的数据到当前实体内。![预览数据](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9068242561/p438642.png)**说明** 添加后的数据回替换原有的数据，请谨慎添加。    |
|    **添加建筑**    |    本操作以添加一个精模建筑数据为例（地标建筑无法自定义手动添加）。单击选中左侧实体列表中的一个精模建筑数据，在右侧空间地图上方单击**添加建筑**按钮，使用鼠标在所需添加模型位置处，按绘制提示说明单击鼠标放置建筑位置标记。![添加建筑](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5286522561/p438142.png)在弹出的**选择模型**对话框中，单击选择您所需样式的模型，并在右侧模型展示区拖动鼠标调整模型到一个合适的视角。单击对话框右下角的**确定**即可完成添加模型。![添加模型对话框](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9353360261/p251035.png)**说明** 添加完成一个精模建筑数据后会排列在左侧精模建筑列表中，您可以查看并修改该精模建筑的详细参数内容。![精模建筑数据](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9353360261/p263918.png)    |
|    **修改地标及精模建筑数据模型参数**    |    本操作以添加一个精模建筑数据为例（地标建筑数据无法更换建筑，只能预览查看位置参数）。单击数据页面内任意一个具体的精模建筑数据，在右侧会弹出当前数据的详情页面，页面内可以查看模型数据的**模型**参数值。在页面中您可以单击**预览**按钮，预览当前精模建筑模型。单击**更换模型**按钮，即可在**选择模型**对话框中重新选择模型进行更换。单击**位置校准**按钮，打开**位置校准**对话框，您可以对精模模型在**二维视图**和**三维预览**页面进行如下操作。**二维视图**![二维视图操作样式图](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9353360261/p252066.png)**移动展示区域**：单击![移动图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0035887161/p251393.jpg)图标，使用鼠标左键单击该灰色区域并拖动即可移动默认精模模型周围的展示区域。**编辑展示区域**：单击![编辑展示区域](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0035887161/p251400.jpg)图标，选中灰色区域边界的四个顶点并拖动即可调整展示区域的范围大小。**绘制模型放置裁剪区域**：单击![裁剪图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0035887161/p251429.jpg)图标，根据绘制说明自定义绘制出一个模型放置区域裁剪面。**删除模型放置裁剪区域**：单击![删除图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0035887161/p252030.jpg)图标，即可删除绘制完成的模型放置裁剪区域。**控制基底数据显隐**：单击**水体**、**绿地**、**道路**和**白模**右侧的![显隐图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0035887161/p252062.jpg)图标，打开这四个基底数据内容；再次单击即可隐藏这四个基底数据。**删除重复区域**：如果打开了**水体**、**绿地**、**道路**或**白模**显示，当您自定义绘制完模型放置裁剪区域后，您可能会看到在地图中原有基底数据会与模型放置区域面重复。地图中会将重复区域以红色标志。您一旦确定模型放置此处，系统会默认将红色重复区域内的基底数据内容删除。![重复区域](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9353360261/p264430.png)**三维预览**：您可以在右侧设置每一次按键调整下的**移动**、**旋转**和**缩放**值；在变换中修改**位置**、**高度**、**角度**和**缩放值**，修改完成后，单击**保存**按钮。![三维预览页](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9068242561/p438651.png)    |
|    **筛选数据**    |    选中某个实体，在展开的列表中单击页面上方的**筛选**按钮，在弹出的**数据筛选**面板中，输入所需数据的属性字段内容关键字，单击**开始筛选**按钮，即可在左侧数据页面中实现精确筛选所需数据内容的功能。单击**重置**按钮，即可清空下方两个筛选字段中的内容。![筛选数据](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0453360261/p242648.png)**说明** 数据筛选暂时仅支持精确筛选。    |
|    **删除数据**    |    选中列表内一个或多个数据，单击上方的![删除](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1035887161/p242687.jpg)图标，在弹出的删除对话框中，单击右下角的**OK**即可在当前实体内删除选中的数据。    |
|    **编辑数据属性并保存**    |    选中列表内一个或多个数据，在展开的面板中自定义编辑不同类型数据的属性值，修改完成后面板上方的**保存 cmd+s**或**批量保存 cmd+s**按钮，完成数据保存；单击数据面板内的**重置**按钮，即可清空下方属性中的内容。图 1\. 单独保存![单独保存](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0453360261/p264517.png)图 2\. 批量保存![批量保存](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0453360261/p242657.png)    |
|    **绘制数据**    |    单击选中左侧实体列表中的具体一个数据（地标建筑和精模建筑类实体数据除外），右侧空间地图上方会显示**开始绘制 (e)**图标。根据您选择的不同类型的数据，可以在地图上自定义绘制不同的数据内容。可供绘制的样式分别为**点**、**线**、**面**三种类型。    |
|    **创建面数据合集**    |    在空间地图中，您可以选择一个**绿地**、**水体**、**遥感影像-道路**、**矢量建筑**实体或**面**通用元素数据项，在数据页面中选中两个数据，地图中会以黄色区域面显示被选中的样式。![创建合集](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9068242561/p439130.png)单击地图右上角的![合集](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0453360261/p242746.jpg)图标，在弹出的合并对话框中，选择下拉列表中某个数据名称，将两个数据合并成以其中一个数据名称命名的全新数据区域，单击右下角的**OK**即可完成创建合集面数据。图 3\. 完成合集创建后面数据样式![合集后的创建样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9068242561/p439132.png)    |
|    **创建面数据交集**    |    在空间地图中，您可以选择一个**绿地**、**水体**、**遥感影像-道路**、**矢量建筑**实体或**面**通用元素数据项，在数据页面中选中两个有一定区域面重叠的数据，地图中会以黄色区域面显示被选中的样式。![交集面区域样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9068242561/p439141.png)单击地图右上角的![交集图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0453360261/p242755.jpg)图标，在弹出的合并对话框中，选择下拉列表中某个数据名称，以选中数据的名称命名交集部分区域新的面数据，单击右下角的**OK**即可完成创建交集面数据。图 4\. 完成交集面数据创建后样式![交集后样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9068242561/p439144.png)    |
|    **创建面数据差集**    |    在空间地图中，您可以选择一个**绿地**、**水体**、**遥感影像-道路**、**矢量建筑**实体或**面**通用元素数据项，在数据页面中选中两个有一定区域面重叠的数据，地图中会以黄色区域面显示被选中的样式。![差集区域样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9068242561/p439148.png)单击地图右上角的![差集图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1453360261/p262904.jpg)图标，在弹出的合并对话框中，选择下拉列表中某个数据名称，以选中数据的名称命名差集部分区域新的面数据，单击右下角的**OK**即可完成创建差集面数据。图 5\. 完成数据差集面区域样式![差集后样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9068242561/p439150.png)    |
|    **剪切面数据**    |    在空间地图中，您可以选择一个**绿地**、**水体**、**遥感影像-道路**、**矢量建筑**实体或**面**通用元素数据项，在数据页面中选中一个数据，地图中会以黄色区域面显示被选中的样式。![面数据展示样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9068242561/p439153.png)单击地图右上角的![裁剪](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2035887161/p242778.jpg)图标，再使用鼠标在所需裁剪的地图位置处，按裁剪提示说明多次单击鼠标完成一个裁剪面数据的绘制，最终显示完成裁剪面后的面数据样式。图 6\. 完成裁剪后的样式![剪贴后样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9068242561/p439158.png)    |

## 项目地图空间范围
