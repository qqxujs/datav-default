通过组件导出成区块功能，您可以将可视化应用中的组件一键生成区块， 并支持配置区块的多种功能。本文介绍在DataV画布编辑器中将组件导出成区块的方法。

## 操作步骤

1. 登录[DataV控制台](https://datav.aliyun.com/)。
2. 在我的可视化页面，单击您创建的可视化应用项目上方的编辑按钮。
3. 在画布编辑器页面，单击图层栏或画布中的某一个组件。
4. 右键单击选择导出成区块，即可打开从选中生成区块对话框。  
图 1\. 导出成区块入口  
![导出成区块入口](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0237591361/p328061.png)  
图 2\. 选中生成区块对话框  
![选中生成区块对话框](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0237591361/p328062.jpg)
5. 在对话框中按要求自定义输入组件生成区块所需的英文名和显示名配置项内容。

**说明** 当组件导出成区块时，组件内的数据除静态数据和API数据类型之外的其他数据源配置都将失效，您需要在组件导出成区块后，重新配置数据内容。
6. 区块创建成功后，您可以单击画布中弹窗的跳转区块编辑器按键，一键跳转至该区块编辑器内，后续您可以对该新建的区块进行区块配置，详情请参见[区块编辑器概览](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/overview-of-block-editor#concept-2059195 "本文为您介绍区块编辑器的结构和功能，帮助您快速了解区块编辑器开发区块。")。![跳转区块编辑器](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0237591361/p328065.jpg)
