  配置区块中组件配置项的显示与隐藏-DataV数据可视化-阿里云

为了避免对外透露出的组件配置项过多，区块设置开放或隐藏某些配置项的设置面板。当您在区块中新增组件后，默认对外隐藏组件样式面板，需在区块配置项设置面板中将隐藏项打开，最终可在区块预览中才能看到该组件的样式设置。本文介绍区块配置项设置面板的功能。

前提条件：已创建一个区块，具体创建区块的方法，请参见[创建空白区块](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/create-a-blank-block#task-2059192 "本文介绍如何创建一个空白区块。")。

## 配置项设置功能说明

在区块编辑器内，您可以在画布中添加一个或多个组件后，单击区块编辑器右上角的![配置项设置入口](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7853813461/p392323.jpg)图标，在配置项设置页签中支持如下样式配置项设置操作。![配置项设置面板](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7853813461/p395322.png)

 

操作

说明

搜索关键字

在输入框内，自定义输入与组件相关的关键字，即可在面板中高亮显示所有相关关键字的配置项内容。![高亮关键配置项](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7853813461/p392324.jpg)

设置组件全量样式显隐

系统默认将组件及所有的配置项隐藏，您可以单击页签内组件右侧的![显隐图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7853813461/p392325.jpg)图标，显示组件的所有样式面板内容。![显示组件全量样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7853813461/p392326.png)

**说明** 再次单击![显隐图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7853813461/p392325.jpg)图标，即可隐藏组件样式面板。

设置组件指定样式配置项显隐

您可以单击左侧树的![打开目录](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8853813461/p392327.jpg)图标，逐级打开组件的样式配置目录树面板。在某个需要设置隐藏的配置项处，单击配置项右侧的![显隐图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7853813461/p392325.jpg)图标，即可隐藏该配置项内容。![控制单个样式显隐](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8853813461/p392329.png)

**说明** 再次单击![显隐图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7853813461/p392325.jpg)图标，即可显示该配置项内容。

定位配置项预览

-   您可以在左侧树面板中， 单击某个配置项右侧的![定位图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8853813461/p392330.jpg)图标，即可在右侧配置预览面板中，高亮定位该配置项。![定位配置项预览样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8853813461/p392332.png)
-   您也可以在配置预览页面中，单击选中一个具体配置项，单击在上方的在左侧树中定位当前配置项处，即可在左侧目录树中快速定位并高亮展示该配置项内容。![反向定位](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8853813461/p392333.png)

## 可视化应用中查看并使用样式面板

在区块编辑器中设置完组件配置项后，您可以在画布编辑器中使用区块时，查看到区块的所有样式信息，具体在可视化应用中配置并查看样式信息的步骤如下。

1.  单击区块编辑器页面右上角的![预览区块图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4123054261/p254797.jpg)图标预览区块。
2.  在预览页面，单击样式面板，可以查看区块内组件的所有样式内容，并可以在样式面板内配置区块不同组件的样式配置项。![查看全量组件样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4090903461/p284179.png)
3.  在区块编辑器中，对配置的组件样式内容进行预览后，单击发布区块，详情请参见[预览区块](https://help.aliyun.com/zh/datav/preview-a-block#task-2062250 "预览区块的目的是为了测试区块的功能可用性，模拟使用时的各种操作。本文介绍如何预览并测试一个配置完成的区块，帮助您及时了解区块的可用性，便于区块的及时修改和完善。")和[发布区块](https://help.aliyun.com/zh/datav/publish-a-block#task-2062363 "本文介绍如何发布一个区块，帮助您升级区块版本，可供后续在可视化应用中使用。")。![预览测试样式功能](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8853813461/p392334.png)
4.  切换到控制台上的我的可视化页面，将鼠标悬浮至一个可视化应用上方，单击编辑进入画布编辑器。
5.  单击![资产列表](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4643285261/p292278.png)图标展开全部资产列表，选择区块模块，在区块列表内可查看到发布后的所有区块内容。
6.  单击所需区块，即可添加该区块到画布编辑器中。
7.  单击选中画布中的区块，在右侧配置面板中选择配置页签，即可查看该区块的配置面板中的所有样式配置信息，您可以参见[组件指南](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/configure-item-description#concept-lsz-bc4-t2b "本文对DataV资产面板中的组件配置项进行了概括性说明，帮助您快速了解各组件的通用和非通用的配置项，并参见对应的文档进行配置。")，修改区块的配置项内容。
