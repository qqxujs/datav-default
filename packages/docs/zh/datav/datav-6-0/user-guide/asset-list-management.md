# 资产列表管理 %{#261225}% 

随着DataV数据可视化产品的搭建物料数量及丰富度的提升，传统的组件概念已经不能覆盖所有搭建物料的概念。本文详细介绍资产列表的含义及其能实现的所有功能。

## 资产包来源

在**全部资产**下拉列表中，展示了您的账号内包括的所有的资产包。这些展示的资产包可分为三大类：![资产分类](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3301927261/p298324.png)

* **官方**：由DataV官方提供的资产包。
* **第三方**：由其他第三方公司及个人提供或者授权的资产包，资产包下的资产左下角会带有![第三方资产](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3301927261/p284585.jpg)图标。![第三方示例](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3301927261/p284592.jpg)
* **自定义**：您自定义的资产包或者由他人授权具有可写权限的资产包，资产包下的资产左下角会带有![自定义图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3301927261/p284587.jpg)图标。![自定义资产](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3301927261/p284593.jpg)

## 资产包切换

在资产库中按照不同资产包的类型进行区别分类，您可以单击资产列表上方的**全部资产**，打开资产包下拉选择列表，单击不同资产包完成资产包内容的切换功能。![单击切换资产包内容](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3301927261/p284553.png)

## 资产包购买

在资产包下拉列表中，对于处于售卖状态的资产包，当您未购买此售卖类型的资产包时，资产包列表右侧会显示**未购买**图标。尚未购买的资产包，无法直接使用，您需要切换到该资产包后，单击**前往购买**，跳转至购买页面购买该资产包。![购买资源包](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3301927261/p298354.png)

## 资产包添加

在资产包下拉列表中，对于处于未添加状态的资产包，资产包列表右侧会显示**未添加**图标。尚未添加的资产包，无法直接使用，您需要单击添加该资产包按照界面提示添加后才能使用。![添加资产包](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3301927261/p298368.png)

## 资产展示区域检索分类

在资产包列表中，单击**全部资产**，即可查看到所有资产中的内容。为了方便您更好地定位查看所需的资产，降低资产检索成本，资产页面重新定义梳理了资产分类的标准。您可以根据分类查找对应资产，也可以在列表上方单击![搜索](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3301927261/p298363.png)图标搜索定位。![4](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5139665761/p556240.jpg)

## 资产内容详情介绍

资产列表作为一个包括所有搭建物料的综合面板，资产列表中的内容包括：组件、区块、素材和模型等。

* 资产面板中**组件**类型的资产包括常规组件类型资产和BI分析组件类型资产，详细使用文档请参见[组件指南](/zh/datav/datav-6-0/user-guide/configure-item-description#concept-lsz-bc4-t2b)。
* 资产面板新增了**区块**类型的资产，详细使用文档请参见[区块管理](/zh/datav/overview-of-blocks#concept-2085792)。
* 资产面板新增了**素材**类型的资产，详细使用文档请参见[素材概述](/zh/datav/datav-6-0/user-guide/material-overview#concept-2142437)。
* 资产面板新增了**模型**类型的资产，详细使用文档请参见[模型概述](/zh/datav/datav-6-0/user-guide/model-overview#concept-2156551)。
* 资产列表面板内所有的资产内容均可添加到画布编辑器中并对资产进行多种操作和配置特定样式，资产在画布编辑器内的操作和配置详情，请参见[资产管理](/zh/datav/datav-6-0/user-guide/search-for-a-widget#task-1392432)。
