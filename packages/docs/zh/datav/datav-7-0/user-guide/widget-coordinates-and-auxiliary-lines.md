 

本文档介绍DataV的组件坐标和辅助线的使用方法，通过组件的坐标和辅助线，您可以在画布中精准地对组件进行定位排布。

## 坐标

在画布中随机添加一个组件，单击该组件即可看到在组件左上角显示的蓝色坐标线（相对于画布左上角的横坐标和纵坐标）。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1162919661/p521063.png)

该线用来定位组件的坐标位置，方便您根据坐标来调整组件位置。

## 辅助线

DataV画布中提供了辅助线的功能，使用方法如下：

1.  移动鼠标至左侧或上方标尺处。
    
2.  选择标尺上某一位置，单击鼠标即可出现辅助线。
    
3.  双击添加的辅助线，可删除该辅助线。
    
4.  在画布左上角，单击辅助线隐藏图标![辅助线图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6457929951/p9213.png)，可隐藏所有辅助线。再次单击显示所有辅助线。 ![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7856871761/p537180.png)
    

## **对齐线**

**说明**

对齐线功能为7.0版本产品新增功能，请根据[新版控制台](https://datav.aliyun.com/v/console/)环境进行操作。

DataV画布中提供了对齐线的功能，使用方法如下：

1.  添加任意两个组件，移动其中一个。
    
2.  当被移动组件在未移动组件**上方**（包括**图层上方**）、**下方**、**左方**和**右方**时（包括**贴紧**和**显示距离**），会出现对齐线和距离标注。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1162919661/p521068.png)![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1162919661/p521069.png)
