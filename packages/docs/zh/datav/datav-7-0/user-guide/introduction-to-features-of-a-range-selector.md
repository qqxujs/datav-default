范围选择器是一款自定义快速选择并生成区域范围数据的小工具，本文介绍范围选择器的详细功能。

## 进入范围选择器

单击[DataV.GeoAtlas地理小工具系列](https://datav.aliyun.com/portal/school/atlas/area%5Fselector)，打开范围选择器工具页面。![{E3D6E567-674F-4FE7-A073-AF2EF64B6263}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1222990571/p965307.png)

## 范围选择器功能详解

| **功能**   | **说明**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 选择地图数据版本 | 单击页面右侧的**数据版本**下拉框，目前支持选择**areas\_v3**地图数据版本。**说明** 当前范围选择器页面内的地图数据来源于高德开放平台。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| 选择区域范围   | 您可以通过以下两种方式选择地图中的区域范围，并展示选中区域范围的内容详情，包括**地名**和**adcode**值。方法一：单击左侧地图中某处区域，即可在右侧选择区域范围处显示选中的区域范围和详情。**说明** 地图支持层级下钻功能，用户可通过多次点击目标区域实现逐级下钻（最小可至县级行政区划），单击下钻区域外的空白区域即可返回上一级视图。方法二：在右侧选择区域范围处，自定义搜索输入区域名称即可选中所需区域并展示区域详情数据。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| 选择数据粒度   | 选择区域范围后，可指定数据粒度并获取对应数据，支持以下格式：**JSON API**：获取区域对应的JSON/GeoJSON数据，单击右侧的![{22919E62-9AB5-4649-9F47-407572122981}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1222990571/p965324.png)图标复制API链接。**其他类型**：如需本地保存数据，您可以选择导出GeoJSON格式的具体数据内容，或SVG格式的范围图形。单击**GeoJSON**数据后的![{22919E62-9AB5-4649-9F47-407572122981}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1222990571/p965324.png)或![{D5F6E569-81C9-4D25-BDEF-A486E58D902B}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1222990571/p965347.png)图标，复制或下载区域数据内容到本地。单击**SVG**右侧的![{D5F6E569-81C9-4D25-BDEF-A486E58D902B}](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1222990571/p965347.png)图标打开预览页面，支持鼠标滚轮缩放地图；单击**下载**按钮可将区域SVG数据保存至本地。 |
