  预览并发布数据看板-DataV数据可视化-阿里云

组件的样式和数据都配置完成后，您可以预览并发布可视化应用，实现应用的在线播放和演示。本文档为您介绍预览并发布可视化应用的方法。

## 操作步骤

1.  单击画布编辑器右上角的预览图标，预览可视化应用。
    
    预览成功后，可按照以下步骤发布可视化应用。
    
2.  单击画布编辑器右上角的发布图标。
    
    ![发布当前大屏](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8449922061/p9452.png)
    
3.  在发布对话框中单击发布大屏。
4.  单击分享链接右侧的复制图标。
    
    ![分享当前大屏链接](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5325446061/p9453.png)
    
5.  打开浏览器，将复制的链接粘贴到地址栏中，即可在线观看发布成功的可视化应用。
