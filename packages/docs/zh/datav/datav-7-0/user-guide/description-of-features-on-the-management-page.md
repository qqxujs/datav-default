 

如您需将自定义的图片、视频、模型等资源添加至数据看板中使用，需先将该类资源创建为DataV-Board 7.0的设计资源。本文为您介绍如何创建设计资源，以及如何执行设计资源的容量查看、分组、详情查看、下载、删除等管理操作，方便您更加高效地管理所有设计资源内容。

## **进入设计资源界面**

1.  登录[DataV-Board 7.0控制台](https://datav.aliyun.com/v/console/)。
    
2.  在左侧导航栏单击**资源管理** > **设计资源**，进入设计资源界面。
    
    在该界面，您可[查看资源容量](#section-g0o-21h-ibe)、[管理资源分组](#section-q6g-gr0-v2g)、[创建设计资源](#84d77ae7b2c11)或执行设计资源的查看详情、下载、重命名等[管理操作](#960836203ci3j)。
    

## 查看资源容量

设计资源按照不同的工作空间进行隔离，各个工作空间的设计资源共享容量池。平台会按照资源的大小进行计量，您可查看当前工作空间下设计资源的已用资源容量和剩余资源容量，以便更好地管理资源内容。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7344883371/p885065.png)

**说明**

-   DataV-Board 7.0不同版本的容量限制存在差异：
    
    -   **企业版资源：**容量为5GB。
        
    -   **专业版资源**：容量为10GB。
        
    -   **尊享版资源**：容量为99GB。
        
-   设计资源按照工作空间隔离，但资源容量按照账号计算。若出现**设计资源**处没有资源，但剩余容量已部分被占用的情况，这可能是由于其他工作空间的设计资源占用了资源容量。
    

## 创建并管理分组

在设计资源界面左侧的**设计资源分组**区域，默认显示全部设计资源，您可对资源进行归类，将同类资源放至同一个分组以便管理。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7344883371/p880059.png)

创建及管理分组的相关操作如下：

-   新建分组：单击**新建分组**，输入分组名称，单击回车键，即可新增一个分组。
    
-   重命名分组：鼠标悬停至目标分组，单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0782082761/p522546.png)图标，可修改分组名称。
    
-   删除分组：鼠标悬停至目标分组，单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0782082761/p522544.png)图标，可删除分组。
    
    **说明**
    
    某个分组删除后，该分组下的设计资源将被移入**未分组**中。
    

## **创建设计资源**

如您需在DataV-Board 7.0中使用自定义图片、视频、模型等资源，则需将该类资源创建为DataV-Board 7.0的设计资源，支持使用如下两种方式创建。

**说明**

目前仅支持将图片、视频、模型等资源文件创建为设计资源。

### **方式一：将本地资源创建为设计资源**

该方式用于将本地的图片、视频或模型上传为设计资源。单击**创建设计资源**，按需选择待上传的资源类型（例如，上传图片），按照界面指引添加资源文件。

示例将本地图片上传为设计资源的步骤如下。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7344883371/p884954.png)

**说明**

支持上传的本地资源文件限制如下：

-   图片：支持`.png`、`.jpg`、`.jpeg`、`.webp`、`.wbmp`、`.svg`、`.gif`、`.tif`、`.bmp`、`.hdr`、`.apng`格式，文件大小不超过5MB；支持批量上传多张图片。
    
-   视频：支持`.mov`、`.mp4`、`.webm`格式，文件大小不超过50MB。
    
-   模型：支持`.gltf`、`.glb`、`.obj`、`.fbx`格式，文件大小不超过50MB。
    

### 方式二：将外部资源创建为设计资源

该方式通过外部资源文件链接，将相关图片、视频或模型创建为设计资源。单击**创建设计资源**，选择**从外部资源创建**，按照界面指引创建资源文件。

**说明**

-   通过外部资源URL链接创建的设计资源文件，无大小限制，且创建的资源不占用设计资源容量。
    
-   模型类设计资源仅专业版及以上版本可用。
    

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3026819071/p771139.png)

配置项介绍如下。

**参数**

**描述**

**地址**

（必填）按需输入公网可访问的外部资源文件URL链接。

**文件名称**

自定义设计资源名称。

**类型**

选择外部资源文件对应的文件类型，支持图片、视频和模型。

**重要**

-   通过设计资源上传模型，目前只支持上传一个完整的嵌入式`.gltf`或`.glb`文件，暂不支持分离的`.gltf`格式（例如，`.gltf+.bin`）。
    
-   若加载的模型出现碎面情况，可能是单mesh的三角面数量超过`4294967296/3≈1431655765`。因此，针对复杂的模型出现碎面问题，建议重新在建模软件中简化面片后再上传。
    

**封面**

用于配置外部资源文件的封面图。可输入所需封面图片的URL地址，也可拖拽或单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7344883371/p885110.png)图标，使用本地图片作为封面图。

**说明**

封面图片的大小不能超过2MB。

设计资源创建后，您可进入看板使用，也可按需执行相关管理操作。

## **管理设计资源**

您可按需执行设计资源的排序、筛选、详情查看、更改分组、重命名等管理操作。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7344883371/p885117.png)

### **全局资源管理**

在设计资源页面右上角，可按需执行如下操作：

-   排序资源：单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7344883371/p885119.png)图标，可按照名称、修改时间、创建时间进行资源排序。默认按照修改时间倒序排序。
    
-   筛选资源：单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7344883371/p885122.png)图标，可按照资源类型筛选资源。默认展示所有类型的资源。
    
-   搜索资源：单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7344883371/p885123.png)图标，可按照名称或URL地址搜索目标资源。
    

### **单个资源管理**

鼠标悬停至目标资源，可按需执行如下操作：

-   查看资源详情：单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7344883371/p885186.png)图标，可查看资源的大小、尺寸、上传时间等详细信息。
    
    在视频和模型类的资源文件详情对话框，支持预览视频或模型文件，并且在详情对话框中支持生成封面截图。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0782082761/p539928.png)
    
    可通过如下方式生成封面：
    
    -   **截取封面**：单击**截取封面**，即可为该文件自动生成一个封面图。
        
        **说明**
        
        -   视频：可在预览时暂停到某一关键帧，截取所需封面图。
            
        -   模型：可在预览时通过鼠标拖拽及滚动鼠标缩放，调整相机视角，选择合适效果，截取所需封面图。
            
        
    -   **上传封面**：单击**上传封面**，即可在本地环境中上传一个图片作为该文件的封面图。
        

-   下载资源：单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7344883371/p885188.png)图标，可下载资源至本地。
    
    **说明**
    
    不支持下载使用外部链接创建的资源。
    
-   更改资源分组：单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7344883371/p885190.png)图标，可更改资源所属的分组。
    
-   获取资源链接：单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7344883371/p885191.png)图标，可获取当前资源的链接。您可将该链接添加至看板中的组件使用，也可使用该链接搜索当前资源。
    
-   删除资源：单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7344883371/p885194.png)图标，可删除当前资源。
    
    **重要**
    
    资源删除后无法恢复，若可视化项目中使用了该资源，将无法正常展现，请谨慎操作。
    
-   重命名资源：单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7344883371/p885204.png)图标，可更改当前资源的名称。
