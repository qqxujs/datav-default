# 素材组件 %{#92358}% 

本文介绍素材类模块内组件的结构及说明。

## **素材组件结构说明**

* 在素材模块内的二级组件面板中，有**视频**、**图标**、**点缀**、**背景图**、**背景框**、**装饰条**和**插画**七个素材类型组件。
* 所有的素材，其组件类型只有**视频**和**单张图片**两类，两类组件的详细配置说明请参见[单张图片](/zh/datav/datav-6-0/user-guide/single-image#concept-eyd-n2l-gfb)和[视频](/zh/datav/datav-6-0/user-guide/video#concept-utj-r2l-gfb)。
* 您可以在画布中添加素材组件内容，按照**单张图片**或**视频**组件配置的使用方法，来自定义修改，包括可配置素材的位置移动、大小缩放、样式配置、数据配置和蓝图配置等，详情请参见[组件指南](/zh/datav/datav-6-0/user-guide/configure-item-description#concept-lsz-bc4-t2b)、[资产管理](/zh/datav/datav-6-0/user-guide/asset-list-management#concept-2088328)。
* 素材组件可在更改关联设计资产功能的配置项时，调用设计资产内容，详情请参见[素材或模型调用设计资产操作示例](/zh/datav/datav-6-0/user-guide/material-or-model-call-design-asset-action-example#task-2094926)。
