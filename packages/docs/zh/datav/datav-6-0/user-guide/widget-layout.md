本文介绍组件布局的操作方法，包括组件拖拽、组件缩放、组件旋转、操作快捷键等，帮助您更加方便快捷地在可视化应用中对组件进行布局。 

|    **操作**    |    **描述**    |
|    ----    |    ----    |
|    拖拽    |    选中一个或多个组件，按住鼠标左键进行拖拽。 选中一个组件：单击某个组件进行选中。选中多个组件：按住键盘上的**Ctrl**键（苹果键盘使用**command**键）后，单击多个组件进行选中。 **说明** 您也可以在左侧的图层列表中，使用同样的方法选中一个或多个组件。    |
|    缩放    |    单选或者多选组件后，将鼠标移动到组件的边界框线附近。出现缩放图标后，按住鼠标左键拖拽。    |
|    旋转    |    选择一个或多个组件，将鼠标移动到组件的四角附近。出现旋转图标后，按住鼠标左键对组件进行旋转。    |
|    对齐    |    选中多个组件后，单击右侧配置面板的**对齐**或**分布**图标。 ![组件对齐样式](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5457929951/p9243.png)支持左、水平居中、右、顶、垂直居中、底等多种对齐方式，以及水平和垂直的自动分布方式。    |
|    设置不透明度    |    选择单一组件或将一个或多个组件成组后，在右侧配置面板上方的基础属性中，单击**不透明度**右侧的**+**或**\-**，或者在输入框中输入透明度值，改变组件或组的不透明度。 ![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5146385761/p557452.png)该功能方便可视化应用设计师按照自己对产品视觉的需求，自由地部署可视化应用。    |
|    组件成组    |    详情请参见[资产成组](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/group-multiple-widgets#task-2375486)。    |
|    组件锁定    |    详情请参见[锁定资产](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/lock-a-widget#task-2376241)。    |
|    组件隐藏    |    详情请参见[隐藏资产](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/hide-a-widget#task-2376266)。    |
|    组件复制    |    详情请参见[复制资产](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/copy-a-widget#task-2376299)。    |
|    组件删除    |    详情请参见[删除资产](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/delete-a-widget#task-2376310)。    |
|    组件图层位置移动    |    详情请参见[组件图层位置移动](https://help.aliyun.com/zh/datav/datav-6-0/user-guide/move-the-layers-of-widgets#concept-krw-sph-t2b)。    |
|    右键菜单    |    在可视化应用画布或者图层栏中，右键单击某个组件。在弹出的菜单中，进行**置顶**、**置底**、**上移一层**、**下移一层**、**复制**、**删除**等操作。**说明** 企业版及以上版本可以使用更多右键菜单功能，包括成组、取消成组、锁定、隐藏等功能。    |
|    缩略图    |    DataV在画布右下角提供了缩略图功能，您可以使用此功能，调整可视化应用在画布中的布局。拉动缩略图下方的**比例尺**，或者单击**+**或**\-**调整画布的大小比例。当可视化应用超出画布大小时，可以拖动缩略图的高亮蓝色边框，定位到可视化应用的任意位置。单击缩略图中右下角的![缩略图图标](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5457929951/p10617.png)图标，可以选择关闭或开启缩略图。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5146385761/p557466.png)**说明** 此功能仅**企业版**及以上版本支持，**基础版**不支持。    |
|    快捷键    |    画布编辑器快捷键： ![快捷键功能介绍面板](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5457929951/p52518.png)开关图层面板：**Ctrl/Cmd + ←** 开关组件面板：**Ctrl/Cmd + ↑** 开关右侧面板：**Ctrl/Cmd + →** 画布缩放到最佳位置：**Ctrl/Cmd + a** 组件快捷键： 分组：**Ctrl + G**解组：**Ctrl + Shift + G**撤回上一步操作：**Ctrl + Z**复制粘贴组件：**Ctrl + C** / **Ctrl + V**    |
