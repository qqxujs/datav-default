 

DataV支持使用模板方式创建PC端数据看板，本文介绍如何使用模板创建PC端数据看板。

## **背景信息**

DataV对新增数据看板的数量有限制，说明如下：

-   **试用版**支持5个数据看板。
    
-   **个人版**支持5个数据看板。
    
-   **企业版**支持20个数据看板。
    
-   **专业版**支持40个数据看板。
    
-   **尊享版**支持1000个数据看板。
    

## **操作步骤**

1.  登录[DataV控制台](https://datav.aliyun.com/v/console/)。
    
2.  在**工作台**页面，将鼠标hover在**创建PC端看板**上，单击**选择模板**。
    
    ![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8023155271/p844891.png)
    
3.  在**创建看板**页面，通过**应用场景**、**设计风格**和**颜色**三种维度筛选出自己需要的模板后，单击**创建看板**。
    
    ![184ed6d8bcfe3ace92c7066519349ea4](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3385833371/p883461.png)
    
4.  在**创建看板**弹窗中，输入**看板名称**，（可选）选择**项目分组**后，单击**创建看板**。
    
    创建成功后，系统会跳转到数据看板编辑页面。
    
    ![5e19644015a628f518e2ede4aa33c72e](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3385833371/p883464.png)
    

**说明**

您也可以在**设计中心 > PC端**，选择空白看板或系统预置模板创建数据看板。
