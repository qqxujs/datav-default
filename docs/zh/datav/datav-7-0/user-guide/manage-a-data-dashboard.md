本文介绍如何管理数据看板，包括预览、发布、查看访问统计、复制、拷屏等功能。

## **预览数据看板**

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，鼠标hover在任一**数据看板**上。
3. 单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7262940761/p516862.png)图标，预览数据看板。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7262940761/p532258.png)

## **发布数据看板**

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，鼠标hover在任一**数据看板**上。
3. 单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7262940761/p516870.png)图标，发布数据看板。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7262940761/p532259.png)
4. 在发布弹窗中，单击发布大屏。

**说明**  
当数据看板**已发布**后，发布页内容会被锁定，每次编辑后可以通过**快照管理**，快速同步发布内容。具体请参见[快照管理](/zh/datav/datav-7-0/user-guide/publish-and-snapshot-management)。

## **查看访问统计**

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，鼠标hover在任一**数据看板**上。
3. 单击![image..png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9932024861/p671330.png)图标，对当前数据看板**查看访问统计**。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7262940761/p532263.png)
4. 在访问统计弹窗中，查看当前数据看板的**PV**值和**UV**值。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3653028961/p730873.png)

## **复制数据看板**

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，选择任一**数据看板**。
3. 单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7262940761/p516881.png)图标下的**复制项目**。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8828297571/p1006915.png)  
系统会自动创建当前数据看板的副本。

## **拷屏数据看板**

您可通过拷屏功能将数据看板拷贝给其他用户使用。

**说明** 

**专业版试用**产品没有**用户识别码**，仅支持将自己的数据看板拷贝给个人版及以上版本的用户，但无法通过拷屏功能接收其他用户的数据看板。版本规格详情，请参见[规格说明](/zh/datav/datav-7-0/product-overview/datav-editions)。

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，选择任一**数据看板**。
3. 单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7262940761/p516881.png)图标下的**拷屏**。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3653028961/p730876.png)
4. 在**拷贝项目给他人**弹窗中，输入**目标用户识别码**后，单击**确定**。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3653028961/p730877.png)

## **进入恢复模式**

**重要** 

当您编写的过滤器有问题 （ 例如死循环导致的看板卡顿 ）、数据源配置有问题（ 例如返回的数据量过大导致的看板卡顿 ）或自定义组件有逻辑问题等其他原因导致数据看板无法编辑时，使用恢复模式进入，可以删除掉带来问题的内容，重新编辑数据看板。

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，选择任一**数据看板**。
3. 单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7262940761/p516881.png)图标下的**进入恢复模式**。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3653028961/p730885.png)
4. 系统自动以恢复模式进入画布编辑器页面，删除不需要的内容后，重新发布即可。

**说明**  
当处于恢复模式时，数据请求及渲染暂停使用。

## **重命名项目**

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，选择任一**数据看板**。
3. 单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7262940761/p516881.png)图标下的**重命名**。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3653028961/p730878.png)
4. 重新输入数据看板的名称后，单击**Enter**键。

## **删除项目**

**说明** 

删除数据看板前需要先关闭发布状态。

1. 登录[DataV控制台](https://datav.aliyun.com/v/console/)。
2. 在**工作台**页面，选择任一**数据看板**。
3. 单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7262940761/p516881.png)图标下的**删除**。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3653028961/p730879.png)
4. 在**删除数据看板**弹窗中，再次单击**删除**。

**重要**  
数据看板一旦删除将无法恢复，请谨慎操作。

单击**数据看板**右侧的**显示全部**，跳转至**全部数据看板**页面，修改排序方式或在搜索框中输入需要查找的数据看板，即可快速定位。且有关数据看板的创建和管理操作均可以使用。也可以单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7262940761/p526464.png)图标或![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7262940761/p526463.png)图标，切换只查看PC端看板或移动端看板。
